# Research Tool Archive

This namespace contains non-daily research, shadow-strategy, and repair-flow
tools. They are retained for reproducibility and forensic review, but they are
not part of the normal Airivo operator surface.

Promotion rule:

- new durable logic belongs in `openclaw/research/` or `openclaw/services/`;
- a tool may return to top-level `tools/` only after it is listed in
  `tools/README.md` and covered by a production-facing test or runbook.
