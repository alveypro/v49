# AIRIVO MAIN SITE HOME EXECUTION SPEC

## Purpose

本文件定义主站首页的落地结构。

## Structure

主站首页固定包含：

- 首屏 Hero
- 产品矩阵区
- CTA 区
- 信任表达区

## Priority

- `/stock` 必须是主系统入口
- `/T12` 只能是治理与边界系统的辅助入口
- 主站负责平台、矩阵、入口与成熟度表达
- 主站不承载业务主判断

## CTA Rules

- 首屏主 CTA 必须导向 `/stock`
- `/T12` 只能作为次级 CTA 或辅助链接出现

## Scope Guard

- 主站不得输出业务主结果结论
- 主站不得复制 `/stock` 主结果结构
- 主站不得复制 `/T12` 治理摘要结构
