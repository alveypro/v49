# STOCK PRIMARY RESULT LANGUAGE SPEC

## Purpose

本文件定义 `/stock` 主结果的语言风格、状态标签与标准句式。

目标：

- 保持结论层短、稳、明确
- 保持解释层克制，不压过主结论
- 保持边界层轻量，不演变成 `/T12` 替身

## Layer Rules

### Conclusion

- 只表达当前业务主判断
- 语言必须短句化、标签化
- 不进入治理摘要语汇
- 不进入主站品牌叙事

### Explanation

- 只表达辅助解释
- 只保留少量、稳定、有来源的说明
- 不把解释层写成制度仲裁文案

### Boundary

- 只保留轻量边界说明
- 明确 `/T12` 才是治理主解释权位置
- 不展开治理详情

## Allowed Status Labels

允许的结论层状态标签来自以下集合：

- 执行状态：`待执行 / 运行中 / 已完成 / 执行失败 / 已取消`
- 审核状态：`审核中 / 已通过 / 未通过 / 已豁免`
- 候选状态：`候选中 / 已入池 / 已驳回 / 已失效`
- 研究状态：`研究中 / 研究完成 / 研究暂停 / 研究终止`

## Standard Expressions

- 空态：`暂缺`
- 异常态：`待补充`
- 降级态：`降级说明`
- 禁用态：优先使用既有 `disabled_reason`
- 失效态：`结果已失效 / 结果已替代 / 结果已驳回 / 结果已失败 / 结果已取消 / 结果已归档`

## Prohibited Terms In Conclusion

以下词汇不得进入 `/stock` 结论层：

- `当前推进状态`
- `当前主要阻断`
- `当前治理备注`
- `Governance Summary`
- 主站品牌叙事词汇，如 `Airivo 是统一母平台`

## Scope Separation

必须留在 `/T12` 的表达：

- 治理摘要
- 推进阻断
- 治理备注
- 治理状态解释

属于主站语言，不得进入 `/stock`：

- 品牌定位
- 平台叙事
- 产品矩阵说明
