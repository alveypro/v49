## 二、四大核心功能（4个Tab）

### 📊 Tab1: 超级回测（统一版）

**用途**：验证选股策略的历史表现

**3步操作**：
```
1. 选择策略：暴涨猎手 / 底部突破猎手 / 终极猎手
2. 设置参数：回测周期180天、样本量800只、持仓5天
3. 点击"开始超级回测"
```

**看什么**：
- ✅ 平均收益（越高越好，>1%优秀）
- ✅ 胜率（越高越好，>52%优秀）
- ✅ 夏普比率（越高越好，>0.3优秀）
- ✅ 信号数量（太少说明条件太严格）

**举例**：
```
回测结果：
- 平均收益：1.2%（优秀）
- 胜率：54.5%（优秀）
- 夏普比率：0.42（优秀）
- 信号数量：523个（合适）

→ 说明这个策略很好，可以在实盘中使用！
```

---

### 🎯 Tab2: 参数优化（增强版）

**用途**：找到最佳持仓天数

**3步操作**：
```
1. 选择优化策略：暴涨猎手 / 底部突破猎手 / 终极猎手 / 所有策略
2. 设置参数：优化周期180天、样本量300只
3. 点击"开始参数优化"
```

**看什么**：
- ✅ 最佳持仓天数（3天/5天/7天/10天中哪个最好）
- ✅ 综合评分（越高越好）
- ✅ 平均收益和胜率

**举例**：
```
优化结果：
- 最佳持仓天数：5天
- 综合评分：8.5分
- 平均收益：1.35%
- 胜率：56.2%

→ 说明用暴涨猎手策略，持仓5天效果最好！
  实盘中就持仓5天卖出。
```

**实用技巧**：
- 选择"所有策略优化"，可以同时找到3个策略的最佳参数
- 对比3个策略的表现，选择综合评分最高的策略

### 🔥 Tab3: 板块扫描分析

**用途**：找出当前热门板块，优先在这些板块选股

**3步操作**：
```
1. 设置统计周期：10天（看最近的热点）
2. 设置最低热度：40分（过滤冷门板块）
3. 点击"扫描热门板块"
```

**看什么**：
- ✅ S级板块（≥80分）：极热板块，最优先
- ✅ A级板块（60-80分）：热门板块，优先
- ✅ B级板块（40-60分）：活跃板块，可选

**举例**：
```
热门板块Top5：
1. 人工智能 - 85分（S级）
2. 半导体 - 78分（A级）
3. 新能源汽车 - 72分（A级）
4. 医药生物 - 55分（B级）
5. 军工 - 48分（B级）

→ 在实盘选股时，优先选人工智能、半导体、新能源汽车板块的股票！
```

**板块的5大作用**：
1. **热门识别** - 资金流向，热门板块股票更容易涨
2. **联动效应** - 同板块多只股票一起涨，不是个股单独行为
3. **轮动预警** - 哪些板块在走强，哪些在走弱
4. **加分机制** - 在热门板块的股票，终极猎手额外加10-15分
5. **风险控制** - 避开走弱的板块，即使个股指标好也要谨慎

推荐结论：

- 最佳风险收益比：终极猎手

  实盘中应该去Tab11（原版系统）使用终极猎手选股。

## 五、常见错误

### ❌ 错误1：只回测不实盘

**问题**：天天回测，就是不去实际选股

**解决**：
- 统一回测版只是分析工具
- 分析完要去原版系统实际选股
- 每周分析一次就够了，不要过度分析

### ❌ 错误2：参数设置极端

**问题**：
- 样本量设置太小（<200只）→ 结果不可信
- 回测周期太短（<60天）→ 数据量不足
- 持仓天数太长（>15天）→ 不符合短线策略

**解决**：
- 样本量：300-1000只
- 回测周期：90-365天
- 持仓天数：3-10天

### ❌ 错误3：忽略板块因素

**问题**：只看个股评分，不看板块

**解决**：
- 先扫描热门板块（Tab3）
- 优先在热门板块选股
- 同等分数下，选热门板块的股票

### ❌ 错误4：频繁切换策略

**问题**：今天用暴涨猎手，明天用终极猎手，后天又换

**解决**：
- 通过策略对比找到最优策略
- 坚持使用至少1个月
- 除非策略明显失效，否则不要频繁切换
