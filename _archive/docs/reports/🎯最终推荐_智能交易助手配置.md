## 🔍 深度分析

### 为什么不推荐60-64分？

```
问题1: 期望值为负(-0.33%)
  → 长期来看会亏钱

问题2: 胜率低(43.4%)
  → 输多赢少

问题3: 盈亏比只有1.00
  → 赚钱和亏钱差不多
  → 没有"赚多亏少"的优势

结论: 必须排除！
```

---

### 为什么65-69分是甜蜜区？

```
优势1: 盈亏比高达1.91
  → 赚的时候平均+5.29%
  → 亏的时候平均-2.78%
  → 一次盈利≈抵消2次亏损

优势2: 期望值为正(+0.56%)
  → 长期来看每笔交易赚0.56%
  → 100笔交易≈赚56%

优势3: 信号充足
  → 29个样本，统计可靠
  → 实战中每天能找到多只候选

结论: 性价比最高！
```

---

### 为什么不全选90+高分？

```
优点: 
  ✅ 胜率50%（最高）
  ✅ 平均收益1.35%（很好）
  ✅ 盈亏比2.03（优秀）

缺点:
  ❌ 信号太少（只有16个样本）
  ❌ 实战中可能每天只有1-2只
  ❌ 错过大量机会

策略:
  → 不是不选，而是"配合使用"
  → 主力仓位：65-75分（3只）
  → 卫星仓位：85+分（2只）
```
