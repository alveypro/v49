## 📝 注意事项

1. **API限流**
   - Tushare Pro有API调用限制
   - 如果提示"无法获取大盘数据"，等待1分钟后重试

2. **日期格式**
   - Tushare要求日期格式为 `YYYYMMDD`
   - 系统已自动处理转换

3. **数据完整性**
   - 非交易日没有数据
   - 系统会自动跳过节假日

## 💡 市场环境判断标准

### 牛市行情 🟢
- 期间涨幅 > 10%
- 预期胜率：60-70%
- 建议：积极操作，优选高分股票

### 上涨趋势 🟡
- 期间涨幅 5-10%
- 预期胜率：55-65%
- 建议：正常操作，精选优质股票

### 震荡行情 🟠
- 期间涨跌 -5% ~ +5%
- 预期胜率：48-55%
- 建议：谨慎选股，控制仓位

### 下跌趋势 🔴
- 期间跌幅 -10% ~ -5%
- 预期胜率：40-48%
- 建议：减少操作，严格止损

### 熊市行情 ⚫
- 期间跌幅 > 10%
- 预期胜率：35-45%
- 建议：空仓观望，规避风险

## 🔍 诊断价值

**为什么这个功能很重要？**

1. **区分策略问题vs市场问题**
   ```
   如果回测期间大盘跌10%，策略胜率44%是正常的
   如果回测期间大盘涨10%，策略胜率44%才是策略问题
   ```

2. **优化回测结果解读**
   ```
   熊市回测 44%胜率 → 策略可能OK
   牛市回测 44%胜率 → 策略需要修复
   ```

3. **指导实盘操作**
   ```
   大盘下跌期间 → 空仓观望
   大盘上涨期间 → 积极选股
   ```
