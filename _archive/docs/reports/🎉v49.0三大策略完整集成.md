## ⚠️ 风险提示

### 系统局限性

1. **历史不代表未来**
   - v4.0虽有56.6%胜率验证
   - 但市场环境会变化
   - 需持续跟踪优化

2. **策略适用性**
   - v5.0和v6.0未经长期验证
   - 建议小仓位测试
   - 逐步建立信心

3. **执行纪律**
   - 必须严格止损
   - 不可过度交易
   - 控制仓位风险

### 操作建议

1. **仓位管理**
   - 单只股票：不超过20%
   - 总仓位：不超过60%
   - 留有现金应对风险

2. **止损纪律**
   - 严格执行-3%止损
   - 或跌破止损价立即止损
   - 不可心存侥幸

3. **止盈策略**
   - 达到+4%或止盈价分批止盈
   - 不可贪心追求最高点
   - 落袋为安

4. **分散投资**
   - 不要全仓一只股票
   - 分散到3-5只
   - 降低单一风险
