## 🔍 第三张图：价格位置说明

### 价格位置是什么？

**计算公式**：
```python
价格位置 = (当前价 - 60日最低价) / (60日最高价 - 60日最低价)
```

**含义**：
- 价格位置 = 0%：股价处于60天最低点
- 价格位置 = 50%：股价处于60天中间位置
- 价格位置 = 100%：股价处于60天最高点

**核心特征标签生成**：
```python
if 价格位置 < 20%:
    标签.append("极低位")
elif 价格位置 < 30%:
    标签.append("低位")

if 量比 > 2.0:
    标签.append("强势放量")
elif 量比 > 1.5:
    标签.append("放量")

if 协同加分 >= 10:
    标签.append("黄金组合")
```

**示例**：
- 股票A：价格位置18% → "极低位"
- 股票B：量比2.3 → "强势放量"
- 股票C：协同加分12分 → "黄金组合"
- 最终显示："极低位 | 强势放量 | 黄金组合"
