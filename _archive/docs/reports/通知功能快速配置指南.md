1. 访问：https://sct.ftqq.com
2. 微信扫码登录
3. 获取您的SendKey（形如：SCT123456789）

#### 第2步：编辑配置文件

```bash
cd /Users/mac/QLIB

```

修改以下内容：

```json
{
  "enabled": true,
  "serverchan": {
    "enabled": true,
    "sendkey": "SCT123456789"  // 填入您的SendKey
  }
}
```

按 `Ctrl+X`，然后 `Y` 保存。

#### 第3步：测试

```python

notifier.test_notification()
```

✅ **完成！现在会推送到您的微信了！**

## 📱 常见邮箱配置

### QQ邮箱（推荐）

```
SMTP服务器: smtp.qq.com
端口: 587
授权码获取: 
  1. 登录QQ邮箱
  2. 设置 → 账户 → 开启SMTP服务
  3. 生成授权码（16位）
```

### 163邮箱

```
SMTP服务器: smtp.163.com
端口: 25
授权码获取: 设置 → POP3/SMTP/IMAP → 开启SMTP
```

### Gmail

```
SMTP服务器: smtp.gmail.com
端口: 587
需要: 开启"允许不够安全的应用访问"
```

## ❓ 常见问题

### Q1: 没收到邮件？

**A: 检查清单**
1. ✅ `notification_config.json` 中 `enabled: true`
2. ✅ 邮箱授权码正确（不是登录密码）
3. ✅ 收件邮箱地址正确
4. ✅ 查看垃圾邮件箱
5. ✅ 查看日志：`tail -f trading_assistant.log`

### Q2: 可以同时用多种通知方式吗？

**A: 可以！** 在配置文件中同时开启多个：

```json
{
  "enabled": true,
  "email": {
    "enabled": true,
    ...
  },
  "serverchan": {
    "enabled": true,
    ...
  },
  "wechat_work": {
    "enabled": true,
    ...
  }
}
```

所有开启的方式都会收到通知！

### Q3: 如何关闭通知？

**A: 三种方式**

1. **临时关闭所有通知**：
```json
{
  "enabled": false,  // 主开关
  ...
}
```

2. **只关闭某种通知**：
```json
{
  "enabled": true,
  "email": {
    "enabled": false  // 只关闭邮件
  }
}
```

3. **删除配置文件**：
```bash
rm notification_config.json
```
