## 🔍 三、维度详细评分逻辑

**核心理念**：识别"即将启动但尚未启动"的底部股票

**逻辑**：成交量温和放大，主力悄悄吸筹（不是暴涨>2.5倍的追高风险）

```python
MACD在零轴下方 且 刚刚金叉（能量柱很小）  → 4分
```


**逻辑**：多条均线粘合，即将发散

### 3.2 维度2：底部特征（20分）⬆️ 提权！

#### 子维度2.1：价格位置（10分）
