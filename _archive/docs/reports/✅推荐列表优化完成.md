## 🎯 优化内容

### 1. 修复评分显示问题 ✅

**问题**: 所有维度评分显示0.0分

**原因**: 字段获取方式错误
```python
# ❌ 错误方式
'潜伏价值': f"{score_result.get('lurking_score', 0):.1f}分"

# ✅ 正确方式
dim_scores = score_result.get('dimension_scores', {})
'潜伏价值': f"{dim_scores.get('潜伏价值', 0):.1f}"
```

## 📝 评级说明

### S级（≥80分）🔥
- **含义**: 完美潜伏机会
- **特征**: 底部+潜伏价值+协同效应强
- **建议**: 重点关注，仓位18-20%
- **预期**: 5天内有较大概率启动

### A级（70-79分）⭐
- **含义**: 优质潜伏标的
- **特征**: 底部特征明显，潜伏价值高
- **建议**: 积极关注，仓位15-18%
- **预期**: 1-2周内有启动可能

### B级（60-69分）💡
- **含义**: 良好机会
- **特征**: 具备一定潜伏价值
- **建议**: 谨慎关注，仓位10-15%
- **预期**: 需要耐心等待

### C级（50-59分）📊
- **含义**: 合格标的
- **特征**: 达到基本标准
- **建议**: 保持观察，仓位5-10%
- **预期**: 可作为备选

## 💰 交易建议

### 入场时机
1. **S级股票**: 可立即建仓50%，确认后加仓
2. **A级股票**: 分批建仓，观察1-2天
3. **B级股票**: 小仓位试探，看表现再定
4. **C级股票**: 观察为主，不急于入场

### 止损策略
- **首要止损**: 跌破止损价（系统计算）
- **次要止损**: 亏损达到-3%
- **时间止损**: 持有5天无表现

### 止盈策略
- **分批止盈**: 达到止盈价分批卖出
- **盈利保护**: 盈利>4%后移动止损
- **时间止盈**: 持有5天后评估是否继续

### 仓位管理
- **单只上限**: 不超过20%
- **总仓位**: 3-5只股票
- **分散原则**: 不同行业、不同评级
