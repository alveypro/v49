## 🆘 技术支持

### 遇到问题？

1. **数据库错误**
   - 检查`permanent_stock_database.db`是否存在
   - 确认表名：`daily_trading_data`和`stock_basic`

2. **评分异常**
   - 检查股票数据是否包含必需字段：`close_price`, `vol`, `pct_chg`
   - 确认数据至少有60条记录

3. **回测慢**
   - 降低`sample_size`参数（如500或800）
   - 使用更短的时间范围（如180天）

4. **结果不理想**
   - 尝试调整阈值（55-65分）
   - 调整止损止盈参数
   - 增加样本量
