## 🎯 v36 新增功能详解

### 1. 🔧 自动参数优化（tab4）

**功能**：
- 自动测试多个参数组合
- 找出最佳的信号强度、评分阈值、持仓天数
- 综合评分 = 平均收益40% + 胜率30% + 夏普比率30%

**使用步骤**：
1. 点击"🔧 自动优化"标签页
2. 选择要测试的参数范围：
   - 信号强度：0.5-0.9
   - 最低评分：60-85
   - 持仓天数：3-15天
3. 点击"🚀 开始自动优化"
4. 查看最佳参数和性能指标

**示例结果**：
```
🏆 最佳参数组合
信号强度: 0.7
最低评分: 75
持仓天数: 7天
信号数量: 245

📊 最佳性能指标
平均收益: 3.45%
胜率: 67.8%
夏普比率: 1.82
```

### 2. 🔄 实时价格更新（tab5）

**功能**：
- 从Tushare获取最新交易数据
- 确保数据库价格是最新的
- 支持快速更新或全量更新

**使用步骤**：

**方式1：快速更新（推荐）**
1. 先在"📊 回测验证"完成选股
2. 切换到"🔄 实时更新"标签页
3. 选择"快速更新（仅当前选股）"
4. 点击"🔄 开始更新"
5. 只更新选出的股票，速度快

**方式2：全量更新**
1. 切换到"🔄 实时更新"标签页
2. 选择"全量更新（所有股票）"
3. 选择更新天数（1-7天）
4. 点击"🔄 开始更新"
5. 更新所有股票，耗时较长

**数据库状态监控**：
- 显示最新交易日期
- 显示股票数量和总记录数
- 提示数据是否为最新

**单只股票查询**：
- 输入股票代码（如：000001.SZ）
- 查看最新价格、涨跌幅
- 确认数据更新时间

## 📞 常见问题

**Q1: 为什么回测结果和预期不一样？**
A: 因为v36使用的是**真实历史数据**，不是理想预测值。真实回测更可靠。

**Q2: 如何确认数据是最新的？**
A: 在"🔄 实时更新"标签页查看"数据库状态"，显示绿色"✅ 数据已是最新"即可。

**Q3: 参数优化需要多久？**
A: 取决于测试组合数量。默认配置约5-10分钟。可以先用少量组合测试。

**Q4: 可以保存优化结果吗？**
A: 当前版本暂不支持。建议截图或记录最佳参数。

**Q5: 为什么更新失败？**
A: 可能原因：
- Tushare积分不足
- 网络连接问题
- 请求频率过高
- 非交易日无数据
