## 🔧 详细修复方案

### 修复1：止损止盈计算逻辑（核心修复！）

**文件**：`comprehensive_stock_evaluator_v4.py`

#### 🔥 修复A：数据顺序问题

```python
# 修复前（第758行）
def _recommend_stop_loss_v4(self, close, ind: Dict) -> Dict:
    current_price = close[-1]  # ❌ 最旧的价格

# 修复后
def _recommend_stop_loss_v4(self, close, ind: Dict) -> Dict:
    # 🔥 重要修复：数据是按日期降序排列的，所以close[0]是最新价格
    current_price = close[0] if len(close) > 0 else close[-1]
```

#### 🔥 修复B：动态止损止盈逻辑

**优化前（固定比例）**：
```python
stop_loss = current_price * 0.95  # 固定-5%
take_profit = current_price * 1.06  # 固定+6%
```

**优化后（根据价格位置动态调整）**：
```python
# 🔥 根据价格位置动态调整止损幅度
if price_position < 0.3:  # 低位
    stop_loss_pct = -0.07  # -7%（宽松，给足空间）
elif price_position < 0.5:  # 中低位
    stop_loss_pct = -0.06  # -6%
elif price_position < 0.7:  # 中位
    stop_loss_pct = -0.05  # -5%
else:  # 高位
    stop_loss_pct = -0.04  # -4%（严格，及时止损）

# 🔥 动态止盈
if price_position < 0.3:  # 低位，空间大
    take_profit_pct = 0.10  # +10%
elif price_position < 0.5:  # 中低位
    take_profit_pct = 0.08  # +8%
elif price_position < 0.7:  # 中位
    take_profit_pct = 0.06  # +6%
else:  # 高位，空间小
    take_profit_pct = 0.05  # +5%
```

**智能逻辑**：
1. **低位股票**：给更宽的止损空间（-7%），更高的止盈目标（+10%）
2. **高位股票**：用更严的止损（-4%），更保守的止盈（+5%）
3. **多重保护**：
   ```python
   # 取最高止损位（最保守）
   stop_loss = max(
       current_price * (1 + stop_loss_pct),  # 根据价格位置
       max(ma20, ma60) * 0.97,               # 均线支撑-3%
       price_min_60 * 1.02                   # 60日最低价+2%
   )
   
   # 确保止损在合理范围
   stop_loss = min(stop_loss, current_price * 0.97)  # 至少-3%
   stop_loss = max(stop_loss, current_price * 0.90)  # 不超过-10%
   ```
