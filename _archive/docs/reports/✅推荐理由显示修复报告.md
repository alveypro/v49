
## 📊 推荐理由结构

### 两部分组成

1. **总体评价** (来自评分器的description)
   - S级: 🔥 完美潜伏机会，重点关注！
   - A级: ⭐ 优质潜伏标的，积极关注
   - B级: 💡 良好机会，谨慎关注
   - C级: 📊 合格标的，保持观察

2. **核心优势** (从dimension_scores中提取Top 3)
   - 自动从8个维度中选出得分最高的3个
   - 格式: 维度名称(分数)
   - 帮助用户理解推荐依据

### 完整示例

```
🔥 完美潜伏机会，重点关注！ | 核心优势: 潜伏价值(18分), 底部特征(15分), 启动确认(13分)
│                           │
└─ 总体评价                 └─ 核心优势（Top 3维度）
```

```python
# Step 1: 获取总体评价
reason_parts = []

# Step 2: 提取核心优势
dim_scores = result.get('dimension_scores', {})
if dim_scores:
    # 按分数排序，取Top 3

full_reason = ' | '.join(reason_parts)
```
