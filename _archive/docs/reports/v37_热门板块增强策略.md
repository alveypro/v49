# 🚀 v37 热门板块增强策略设计

## 💡 核心理念

**在热门板块和即将热门的板块中，用量价关系找暴涨股票**

### 为什么这个策略更有效？

1. **资金聚集效应**
   - 热门板块有大量资金流入
   - 板块内个股容易形成共振
   - 龙头股带动效应明显

2. **情绪溢出效应**
   - 一个板块热起来，相关股票都会受益
   - 市场关注度高，成交活跃
   - 更容易出现短期爆发

3. **提前布局优势**
   - 识别"即将热门"的板块
   - 在资金大规模进入前布局
   - 获得更大收益空间

## 📊 技术实现方案

### 1. 板块热度计算

```python
def calculate_sector_heat(sector_stocks):
    """
    计算板块热度
    """
    # 1. 平均涨幅
    avg_change = sector_stocks['pct_chg'].mean()
    
    # 2. 涨停数量
    limit_up_count = (sector_stocks['pct_chg'] >= 9.9).sum()
    
    # 3. 成交额增幅
    vol_change = sector_stocks['vol_ratio'].mean()
    
    # 4. 板块强度（超过大盘的比例）
    outperform_ratio = (sector_stocks['pct_chg'] > market_change).mean()
    
    # 综合评分
    heat_score = (
        avg_change * 30 +           # 涨幅权重30%
        limit_up_count * 20 +       # 涨停数权重20%
        vol_change * 25 +           # 成交量权重25%
        outperform_ratio * 25       # 强度权重25%
    )
    
    return heat_score
```

### 2. 即将热门识别

```python
def identify_emerging_sectors(df, days=20):
    """
    识别即将热门的板块
    """
    for sector in all_sectors:
        sector_data = df[df['industry'] == sector]
        
        # 信号1：缩量蓄势
        recent_vol = sector_data[-3:]['vol'].mean()
        historical_vol = sector_data[-20:-3]['vol'].mean()
        vol_shrink = recent_vol / (historical_vol + 1)
        
        # 信号2：开始放量
        latest_vol = sector_data[-1:]['vol'].mean()
        vol_surge = latest_vol / (recent_vol + 1)
        
        # 信号3：价格未大涨
        price_change = sector_data[-5:]['pct_chg'].mean()
        
        # 信号4：有龙头启动
        strong_stocks = sector_data[sector_data['signal_strength'] > 70]
        
        if vol_shrink < 0.8 and vol_surge > 1.3 and price_change < 3 and len(strong_stocks) > 0:
            # 这是一个即将热门的板块！
            return True
```

### 3. 板块+量价组合策略

```python
def sector_volume_strategy(df):
    """
    板块+量价组合策略
    """
    # Step 1: 识别热门和即将热门的板块
    hot_sectors = get_hot_sectors(df)          # 热门板块
    emerging_sectors = get_emerging_sectors(df) # 即将热门
    
    target_sectors = hot_sectors + emerging_sectors
    
    # Step 2: 在这些板块中应用量价分析
    for sector in target_sectors:
        sector_stocks = df[df['industry'] == sector]
        
        # 应用v36的量价信号识别
        signals = identify_volume_price_signals(sector_stocks)
        
        # Step 3: 综合评分
        for stock in signals:
            # 板块热度加分
            sector_score = hot_sectors.get(sector, 0)
            
            # 个股信号强度
            stock_score = stock['signal_strength']
            
            # 综合得分
            final_score = stock_score * 0.6 + sector_score * 0.4
            
            stock['final_score'] = final_score
    
    return sorted_stocks_by_final_score
```
