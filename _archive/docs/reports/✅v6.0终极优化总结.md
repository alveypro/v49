### 3. 超强大脑优化：智能资金流向评分

**核心思想**：趋势 > 绝对值，加速度 > 规模

#### 🔥 三大智能评估

**1. 加速度检测**
```python
# 检测今日资金流入是否加速
today_net = money_flow.get('today_net', 0)
is_accelerating = today_net > net_mf / 3  # 今日占3日总量1/3以上

# 评分时优先考虑加速度
if consecutive_days >= 3:
    if net_mf > 20000 or is_accelerating:  # 大额或加速都给高分
        score += 15
```

**2. 相对强度评估**
```python
# 计算超大单占比
elg_ratio = abs(elg_net / lg_net) if lg_net != 0 else 0

# 主力强势判断
if elg_net > 5000:
    if elg_ratio > 0.5:  # 超大单占比>50%，主力强势
        score += 8
    else:
        score += 6
```

**3. 连续性评估**
```python
# 连续性比金额大小更重要
if consecutive_days >= 3:
    # 连续3天，即使金额不大也给高分
    if net_mf > 10000:
        score += 12
    elif net_mf > 5000:
        score += 9
    else:
        score += 7  # 连续3天，最少也给7分
```
