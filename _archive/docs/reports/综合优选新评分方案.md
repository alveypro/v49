# 🏆 综合优选新评分方案

## 📊 新的4维评分体系（总分90分）

### 删除的指标
- ❌ 主力吸筹（15分）
- ❌ 底部特征（10分）

### 保留并优化的指标

#### 【维度1】量价配合（30分）🔥 - 提高5分
**原25分 → 现30分**

**评分标准**：
- 强势放量上涨（量比>2.0 + 3日涨>3%）：30分
- 放量上涨（量比>1.8 + 3日涨>2%）：24分
- 温和放量上涨（量比>1.5 + 3日涨>1%）：18分
- 小幅放量上涨（量比>1.2 + 3日涨>0%）：12分

**一票否决**：放量下跌直接0分

**权重占比**：33.3%

---

#### 【维度2】均线多头（25分）⭐ - 提高5分
**原20分 → 现25分**

**评分标准**（5/15/30天）：
- 完美多头排列（MA5>MA15>MA30 且价格>MA5）：25分
- 强势多头（MA5>MA15 且价格>MA5）：20分
- 中期多头（MA5>MA15）：15分
- 站上15日线（价格>MA15）：10分
- 站上5日线（价格>MA5）：6分

**权重占比**：27.8%

---

#### 【维度3】MACD趋势（30分）⭐⭐⭐ - 大幅提高
**原15分 → 现30分**

**核心改进**：考虑三个方向（DIF、DEA、MACD柱）

**评分标准**：

1. **完美三向上（30分）** - 最强信号
   - DIF向上（DIF[-1] > DIF[-2]）
   - DEA向上（DEA[-1] > DEA[-2]）
   - MACD柱向上（MACD[-1] > MACD[-2]）
   - 且DIF > DEA（金叉状态）
   - 💡 三个箭头都向上！

2. **0轴附近金叉+双向上（25分）** - 提高权重
   - 刚金叉（DIF[-2]<=DEA[-2] && DIF[-1]>DEA[-1]）
   - DIF和DEA在0轴附近（|DIF|<0.5）
   - DIF向上 且 DEA向上
   - 💡 位置好+方向好

3. **底部金叉+单向上（20分）**
   - 底部金叉（DIF<0 && DEA<0）
   - DIF向上 或 DEA向上
   - 💡 底部反转信号

4. **刚金叉（15分）**
   - DIF[-2]<=DEA[-2] && DIF[-1]>DEA[-1]
   - 💡 刚出现金叉

5. **金叉持续（10分）**
   - DIF > DEA 且 DIF > 0
   - 💡 已在0轴上方

6. **DIF>DEA（5分）**
   - 只满足DIF > DEA
   - 💡 基础金叉状态

**权重占比**：33.3%

---

#### 【维度4】涨停基因（5分）- 保持不变
**近5天内有涨停记录**

**评分标准**：
- 近5天有2个以上涨停：5分
- 近5天有1个涨停：3分

**权重占比**：5.6%

## 📈 MACD三向判断逻辑

### 判断方法
```python
# DIF向上
dif_up = dif[-1] > dif[-2]

# DEA向上  
dea_up = dea[-1] > dea[-2]

# MACD柱向上
macd_bar = dif - dea
macd_up = macd_bar[-1] > macd_bar[-2]

# 三向都向上
all_up = dif_up and dea_up and macd_up
```

### 评分逻辑
1. **完美三向上（30分）**：
   - 三个箭头都向上 ✅
   - 且DIF > DEA（金叉状态）✅
   - 这是最强的趋势信号

2. **0轴附近金叉+双向上（25分）**：
   - 刚金叉（突破）✅
   - 在0轴附近（位置好）✅
   - DIF和DEA都向上（方向好）✅

3. **底部金叉+单向上（20分）**：
   - 底部位置（DIF/DEA<0）✅
   - 至少一个向上 ✅

## 💡 优化理由

### 为什么删除主力吸筹？
- 主力吸筹本质上是量价配合的一部分
- 连续放量+价格稳定 = 量价配合的延伸
- 删除后简化逻辑，不影响选股质量

### 为什么删除底部特征？
- 过度强调底部会错过中途启动的股票
- 均线和MACD已经能反映趋势位置
- 删除后更关注趋势和动能

### 为什么MACD提高到30分？
- MACD是趋势反转的最强信号
- 三个方向都向上 = 趋势、动能、持续性都好
- 提高权重能更好捕捉强势股

### 为什么考虑三个方向？
- DIF向上：快线向上，趋势加速
- DEA向上：慢线向上，趋势确认
- MACD柱向上：柱子变长，动能增强
- **三者都向上 = 完美的趋势加速状态**
