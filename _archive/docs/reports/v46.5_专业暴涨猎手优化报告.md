## 📊 核心优化内容

### 🔥 关键突破：区分放量上涨 vs 放量下跌

这是本次优化的**最核心改进**！之前的版本只看成交量是否放大，但**没有区分涨跌方向**。

#### ❌ 旧版本问题：
```python
# 只看放量，不区分涨跌
if vol_ratio > 2.5:
    score += 30  # 无论涨跌都加分 ❌
```

#### ✅ 新版本优化：
```python
# 🔥 必须是放量上涨！
if vol_ratio > 2.0 and price_chg_3d > 0.03:  # 放量 + 上涨
    score_volume_price = 25  # 主力吸筹 ✅
elif vol_ratio > 2.0 and price_chg_3d < -0.02:  # 放量 + 下跌
    score_volume_price = -10  # 主力出货，负分！❌
```

**专业分析**：
- ✅ **放量上涨** = 成交量↑ + 价格↑ → 主力吸筹，买入信号
- ❌ **放量下跌** = 成交量↑ + 价格↓ → 主力出货，必须规避！
- 💡 这是量价分析的基础，却常被忽略

## 💎 十维专业评分系统（总分100分）

### 1️⃣ 量价配合（25分）🔥
**核心逻辑**：
- 强势放量上涨（vol_ratio>2.0 且 price_chg>3%）：**25分**
- 放量上涨（vol_ratio>1.8 且 price_chg>2%）：**20分**
- 温和放量上涨（vol_ratio>1.5 且 price_chg>1%）：**15分**
- ❌ 放量下跌（vol_ratio>2.0 且 price_chg<-2%）：**-10分**

**专业意义**：量价齐升是最可靠的买入信号

---

### 2️⃣ 底部特征（20分）
**核心逻辑**：
- 底部位置（<25%）+ 前期缩量：**20分**
- 底部位置（<25%）：**15分**
- 相对低位（<35%）：**10分**

**专业意义**：
- 低位 = 主力成本低，拉升空间大
- 前期缩量 = 筹码集中，洗盘充分

---

### 3️⃣ 主力吸筹（15分）
**核心逻辑**：
- 连续温和放量 + 价格稳定 + 适中换手：**15分**
- 连续温和放量 + 价格稳定：**10分**
- 温和放量：**5分**

**识别特征**：
- 连续2-3天成交量>均值1.2倍
- 价格不跌（pct_chg ≥ -0.5%）
- 相对换手率1.2-2.0倍（活跃但不过度）

**专业意义**：识别主力建仓行为，区别于出货

## 🎓 专业知识总结

### 量价关系的本质：

1. **量价齐升**（放量上涨）
   - 成交量放大，价格上涨
   - 说明：买盘力量强，主力吸筹
   - 操作：**买入信号** ✅

2. **量增价跌**（放量下跌）
   - 成交量放大，价格下跌
   - 说明：卖盘力量强，主力出货
   - 操作：**卖出信号** ❌

3. **量缩价稳**（缩量整理）
   - 成交量萎缩，价格稳定
   - 说明：浮筹清理，洗盘充分
   - 操作：**观察等待**

4. **缩量后再放量**（洗盘后启动）
   - 先缩量回调，再放量上涨
   - 说明：洗盘完成，主力拉升
   - 操作：**最佳买点** 🔥

### 主力行为识别：

1. **吸筹阶段**
   - 低位震荡
   - 温和放量
   - 价格不跌

2. **洗盘阶段**
   - 价格回调
   - 成交量萎缩
   - 浮筹清理

3. **拉升阶段**
   - 放量突破
   - 量价齐升
   - 连续上涨

4. **出货阶段**
   - 高位震荡
   - 放量滞涨
   - 放量下跌
