## 🎓 算法原理简介

我的**筹码分布计算系统**采用以下算法：

### 1. 指数衰减法
```
近期成交的筹码权重更大，远期的逐渐衰减
权重 = e^(-衰减系数 × 天数)
```

### 2. 成本计算
```
平均成本 = Σ(价格 × 筹码量 × 权重) / Σ(筹码量 × 权重)
```

### 3. 获利比例计算
```
获利比例 = (当前价格以下的筹码量 / 总筹码量) × 100%
```
