#### 新增方法2：_backtest_v7_with_adaptive_system

**方法签名**:
```python
def _backtest_v7_with_adaptive_system(
    self, 
    df: pd.DataFrame, 
    sample_size: int, 
    holding_days: int, 
    score_threshold: float
) -> dict:
```

**核心逻辑**:

1. **重置v7.0缓存**
   ```python
   self.evaluator_v7.reset_cache()
   ```

2. **股票采样**
   ```python
   stock_list = df[['ts_code', 'name', 'industry']].drop_duplicates()
   if len(stock_list) > sample_size:
       stock_list = stock_list.sample(n=sample_size, random_state=42)
   ```

3. **逐股票、逐时点回测**
   ```python
   for idx, stock_row in stock_list.iterrows():
       ts_code = stock_row['ts_code']
       stock_name = stock_row['name']
       industry = stock_row['industry']
       
       # 获取该股票历史数据
       stock_data = df[df['ts_code'] == ts_code].copy()
       
       # 遍历历史数据
       for i in range(60, len(stock_data) - holding_days):
           # 当前时点数据（不包含未来）
           current_data = stock_data.iloc[:i+1].copy()
           
           # 使用v7.0评分器评分
           eval_result = self.evaluator_v7.evaluate_stock_v7(
               current_data, ts_code, industry
           )
           
           if eval_result['success'] and eval_result['final_score'] >= score_threshold:
               # 计算未来收益
               buy_price = stock_data.iloc[i]['close']
               sell_price = stock_data.iloc[i + holding_days]['close']
               future_return = (sell_price / buy_price - 1) * 100
               
               # 记录信号
               backtest_results.append({...})
               break  # 每只股票只记录一次
   ```

4. **计算统计指标**
   ```python
   stats = {
       'total_signals': 信号总数,
       'win_rate': 胜率,
       'avg_return': 平均收益,
       'max_return': 最大收益,
       'max_loss': 最大亏损,
       'sharpe_ratio': 夏普比率,
       'sortino_ratio': Sortino比率,
       'max_drawdown': 最大回撤,
       'profit_loss_ratio': 盈亏比,
       'avg_holding_days': 平均持仓天数
   }
   ```

5. **返回结果**
   ```python
   return {
       'success': True,
       'strategy': 'v7.0 终极智能版',
       'stats': stats,
       'details': backtest_df
   }
   ```

## 🎯 v7.0回测的特殊性

### 与v4/v5/v6的主要区别

| 特性 | v4/v5/v6回测 | v7.0回测 |
|------|--------------|----------|
| **评分器参数** | 只需stock_data | 需要stock_data + ts_code + industry |

| **行业因素** | 不考虑 | 自动识别热门行业，热门加分 |
| **市场感知** | 不考虑 | 识别市场环境，自适应调整 |

## ⚠️ 重要提示

### 回测vs实盘的差异

```
⚠️ 回测无交易成本（手续费、印花税、滑点）
⚠️ 回测假设立即成交（实盘可能有延迟）
⚠️ 回测使用收盘价（实盘可能无法以收盘价成交）
⚠️ 回测无情绪干扰（实盘可能无法严格执行）
```

**建议**:
- 回测胜率打8折
- 回测收益打7折
- 加入2-3%的交易成本

**例如**:
```
回测：胜率65%，平均收益3.5%
实盘预期：胜率52%，平均收益2.0%（扣除2%成本后）
```
