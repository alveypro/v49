## 🔧 使用方法

### 方法1：命令行模式（立即可用）

```bash
cd /Users/mac/QLIB
python trading_assistant.py
```

### 方法2：Python交互模式

```python
from trading_assistant import TradingAssistant

# 创建助手
assistant = TradingAssistant()

# 1. 每日选股
recommendations = assistant.daily_stock_scan(top_n=5)
for rec in recommendations:
    print(f"{rec['stock_name']}: {rec['score']:.1f}分")

# 2. 添加持仓（手动买入后记录）
assistant.add_holding(
    ts_code='600519.SH',
    buy_price=1800.00,
    quantity=100,
    score=75
)

# 3. 更新持仓（获取最新价格）
assistant.update_holdings()

# 4. 检查止盈止损
alerts = assistant.check_stop_conditions()
for alert in alerts:
    print(alert['message'])

# 5. 卖出持仓
assistant.sell_holding(
    ts_code='600519.SH',
    sell_price=1900.00,
    reason='止盈6%'
)

# 6. 生成报告
report = assistant.generate_daily_report()
print(report)

# 7. 查看统计
stats = assistant.get_statistics()
print(f"胜率: {stats['win_rate']*100:.1f}%")
```

## 📊 数据库说明

### 自动创建的数据库：`trading_assistant.db`

包含4个表：

1. **holdings**（持仓表）
   - 股票代码、名称
   - 买入日期、价格、数量
   - 当前价格、市值
   - 盈亏金额、盈亏比例

2. **trade_history**（交易记录）
   - 买入/卖出记录
   - 交易价格、数量
   - 盈亏统计

3. **daily_recommendations**（每日推荐）
   - 推荐日期、股票
   - 评分、价格、理由

4. **config**（配置表）
   - 策略参数
   - 止盈止损设置

## ⚙️ 配置参数

默认参数（可自定义）：

```python
策略：v4.0长期稳健版
最低评分：65分
市值区间：100-500亿
推荐数量：5只
单只仓位：20%
最大仓位：80%
止盈：6%
止损：4%
持仓周期：5天
```

修改配置：
```python
assistant.update_config('min_score', '70')  # 提高到70分
assistant.update_config('take_profit_pct', '0.08')  # 止盈8%
```
