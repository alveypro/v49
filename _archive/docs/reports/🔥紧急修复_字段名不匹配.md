# 🔥 紧急修复：字段名不匹配导致评分失败

**发现时间**: 2024-12-24  
**问题严重性**: ⚠️ 致命！导致所有股票评分失败

---

## ❌ 问题根源

### 查询返回的字段（第4592行）
```python
SELECT trade_date, close_price as close, vol as volume, pct_chg
FROM daily_trading_data
```

返回字段：
- `close` ❌
- `volume` ❌  
- `pct_chg` ✅

### 评分器需要的字段（evaluator_v4.py 第69行）
```python
required_cols = ['close_price', 'vol', 'pct_chg']
```

需要字段：
- `close_price` ✅
- `vol` ✅
- `pct_chg` ✅

### 问题
**查询用了别名 `as close` 和 `as volume`，但评分器需要原始字段名！**

---

## ✅ 解决方案

### 修改查询语句，不使用别名

**修改前**（第4592行）:
```python
data_query = """
    SELECT trade_date, close_price as close, vol as volume, pct_chg
    FROM daily_trading_data
    WHERE ts_code = ?
    ORDER BY trade_date DESC
    LIMIT 120
"""
```

**修改后**:
```python
data_query = """
    SELECT trade_date, close_price, vol, pct_chg
    FROM daily_trading_data
    WHERE ts_code = ?
    ORDER BY trade_date DESC
    LIMIT 120
"""
```

---

## 🎯 这就是为什么之前能工作的原因

之前的版本可能：
1. 查询没有使用别名
2. 或者评分器接受 `close` 和 `volume` 字段
3. 某次修改时不小心改错了
