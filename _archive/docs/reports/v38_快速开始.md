# 🎯 v38 快速开始指南

## 🚀 5分钟学会使用v38

### 第一步：启动系统（1分钟）

```bash
cd /Users/mac/QLIB
bash start_v38.sh
```

打开浏览器访问：`http://localhost:8503`

---

### 第二步：寻找萌芽期板块（2分钟）

1. 点击【板块生命周期】Tab
2. 选择一个板块（如"新能源"）
3. 点击"分析板块生命周期"
4. 查看结果：
   - ✅ 如果显示"萌芽期"（评分70+） → 进入第三步
   - ❌ 如果显示其他阶段 → 换一个板块继续分析

**萌芽期特征：**
- 🟢 成交量萎缩
- 🟢 价格横盘
- 🟢 评分70-80分
- 🟢 操作建议："分批买入"

---

### 第三步：识别龙头股（1分钟）

1. 点击【龙头股狙击】Tab
2. 选择刚才的萌芽期板块
3. 点击"识别龙头股"
4. 查看结果：
   - 🏆 龙头股（评分80+） → 重点买入
   - ⭐ 跟随股（评分60-80） → 可以配置

---

### 第四步：三周期确认（1分钟）

1. 点击【三周期共振】Tab
2. 选择同一个板块
3. 点击"分析周期共振"
4. 查看结果：
   - 🚀 三线金叉（评分90+） → 确定性机会
   - 💎 低吸机会（评分80+） → 分批买入
   - ❌ 反弹陷阱（评分30-） → 坚决回避

---

### 第五步：开始操作！

**如果找到：萌芽期板块 + 龙头股 + 三线金叉**

🎯 **恭喜！这是最佳买入机会！**

**操作建议：**
1. 分3-5天分批建仓
2. 总仓位20-30%
3. 目标收益+50%
4. 止损-8%

## ⚠️ 重要提示

### 三要素缺一不可

❌ 只有萌芽期，没有龙头股 → 不买
❌ 只有龙头股，没有萌芽期 → 不买
❌ 只有三线金叉，不是萌芽期 → 不买

✅ 萌芽期 + 龙头股 + 三线金叉 → 买！

- 萌芽期：-8%必须止损

### 及时止盈
