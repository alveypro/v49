# ⚡ 立即执行：清理缓存并重启系统

## 问题原因
您遇到的错误是因为 **Streamlit缓存了旧代码**。文件已经修复，但系统仍在使用缓存的旧版本。

## 解决方案

### 方法1：使用Python脚本（推荐）

打开终端，执行：

```bash
cd /Users/mac/QLIB
/opt/miniconda3/bin/python restart_v49_now.py
```

### 方法2：手动操作

#### 步骤1: 停止当前系统
```bash
pkill -f "streamlit run 终极量价暴涨系统"
```

#### 步骤2: 清理缓存
```bash
cd /Users/mac/QLIB
rm -rf __pycache__
rm -rf ~/.streamlit/cache
rm -rf .streamlit/cache
find . -name "*.pyc" -delete
```

#### 步骤3: 重新启动
```bash
/opt/miniconda3/bin/streamlit run 终极量价暴涨系统_v49.0_长期稳健版.py --server.port 8501
```

### 方法3：在浏览器中强制刷新

如果系统已经在运行，只需：

1. 在浏览器中访问 http://localhost:8501
2. 按 **Ctrl+Shift+R** (Windows/Linux) 或 **Cmd+Shift+R** (Mac)
3. 这会清除浏览器缓存并重新加载页面

## 验证修复

启动后，进入Tab12（v4.0策略），点击"开始扫描"。

如果看到以下内容，说明修复成功：
- ✅ 扫描正常开始

1. 完全关闭浏览器
2. 停止Streamlit进程
3. 清理所有缓存
4. 重新启动
