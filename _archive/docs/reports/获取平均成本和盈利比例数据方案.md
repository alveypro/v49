## 🧮 方案三：算法估算（推荐，可自己实现）

### 原理
通过历史成交数据计算筹码分布，估算平均成本和获利比例

### 实现步骤

#### 1. **数据准备**（使用 Tushare）
```python
import tushare as ts
import pandas as pd
import numpy as np

# 获取历史日线数据（建议取近 1-2 年）
ts.set_token('你的token')
pro = ts.pro_api()

# 获取日线数据
df = pro.daily(ts_code='000001.SZ', 
               start_date='20230101', 
               end_date='20251218')
```

#### 2. **筹码分布算法**
```python
def calculate_chip_distribution(df, decay_days=120):
    """
    计算筹码分布
    
    参数:
    - df: 日线数据 (包含 close, vol, amount)
    - decay_days: 筹码衰减周期（默认 120 天）
    
    返回:
    - 平均成本
    - 获利比例
    """
    # 按日期排序
    df = df.sort_values('trade_date')
    
    # 计算每日的成交额均价
    df['avg_price'] = df['amount'] * 1000 / (df['vol'] * 100)
    
    # 初始化筹码分布字典
    chips = {}
    
    # 遍历每一天
    for idx, row in df.iterrows():
        date = row['trade_date']
        price = row['avg_price']
        volume = row['vol']
        
        # 筹码衰减（模拟换手）
        for p in list(chips.keys()):
            chips[p] *= 0.99  # 每天衰减 1%
            if chips[p] < volume * 0.001:  # 清理小筹码
                del chips[p]
        
        # 添加当日筹码
        if price in chips:
            chips[price] += volume
        else:
            chips[price] = volume
    
    # 计算平均成本
    total_volume = sum(chips.values())
    avg_cost = sum(p * v for p, v in chips.items()) / total_volume
    
    # 获取当前价格（最新收盘价）
    current_price = df.iloc[-1]['close']
    
    # 计算获利比例
    profit_volume = sum(v for p, v in chips.items() if p < current_price)
    profit_ratio = profit_volume / total_volume * 100
    
    return avg_cost, profit_ratio, chips
```

#### 3. **使用示例**
```python
# 获取数据
df = pro.daily(ts_code='000001.SZ', 
               start_date='20230101', 
               end_date='20251218')

# 计算
avg_cost, profit_ratio, chips = calculate_chip_distribution(df)

print(f"平均成本: {avg_cost:.2f}")
print(f"获利比例: {profit_ratio:.2f}%")
```
