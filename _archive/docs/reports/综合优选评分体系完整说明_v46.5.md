## 📐 评分计算公式

```python
最终得分 = 基础得分 + 协同加分 - 风险扣分 + 行业加分

其中：
- 基础得分 = 量价(25) + 均线(20) + MACD(25) + 底部(15) + 主力(10) + 涨停(5)
- 协同加分 = 0~20分 (识别黄金组合)
- 风险扣分 = 0~41分 (系统性风险惩罚)
- 行业加分 = 0~5分 (热门行业奖励)
- 最终限制在 [0, 100] 区间
```

## 📊 6维基础评分体系（100分）

### 🔥 维度1：量价配合（25分）

**计算逻辑**：
- 量比 = 近3天均量 / 历史20天均量
- 价格变化 = 近5天涨跌幅
- 价格位置 = (当前价 - 60日最低) / (60日最高 - 60日最低)

**评分规则**：
```python
低位(位置<30%) + 强势放量(量比>2.0) + 上涨>3%  → 25分 ⭐满分
低位(位置<40%) + 放量(量比>1.5) + 上涨>2%     → 20分
放量(量比>1.3) + 上涨>0%                    → 15分
温和放量(量比>1.1) + 上涨>0%                → 10分
放量(量比>1.5) + 下跌<-2%                   → 0分  ⚠️一票否决
```

**代码位置**：`_score_volume_price()` 函数（第1761-1794行）

---

### 📈 维度2：均线多头（20分）

**计算逻辑**：
- MA5 = 近5天收盘价均值
- MA10 = 近10天收盘价均值  
- MA20 = 近20天收盘价均值
- MA60 = 近60天收盘价均值

**评分规则**：
```python
MA5 > MA10 > MA20 > 0 + 涨幅>2%  → 20分 ⭐满分
MA5 > MA10 > MA20 > 0           → 18分
MA5 > MA10 > 0                  → 15分
MA5 > MA20 或 MA10 > MA20       → 10分
MA5 > 0                         → 5分
```

**代码位置**：`_score_ma_trend()` 函数（第1796-1816行）

---

### 🎯 维度3：MACD趋势（25分）

**计算逻辑**：
- DIF = EMA12 - EMA26
- DEA = DIF的9日EMA
- 柱状线 = DIF - DEA
- 金叉 = DIF上穿DEA（前一天DIF≤DEA，当天DIF>DEA）

**评分规则**：
```python
三向上(DIF↑ DEA↑ 柱↑) + 零轴上    → 25分 ⭐满分
三向上 + 底部金叉                 → 22分
三向上                          → 20分
零轴上金叉 + 双向上               → 20分
底部金叉 + 双向上                → 18分
普通金叉                        → 15分
持续金叉 + 双向上                → 15分
持续金叉                        → 10分
准备金叉（接近交叉）              → 8分
```

**代码位置**：`_score_macd_trend()` 函数（第1818-1863行）

---

### 💎 维度4：底部特征（15分）

**计算逻辑**：
- 价格位置 = (当前价 - 60日最低) / (60日最高 - 60日最低)
- 波动率 = std(近20日收盘价) / mean(近20日收盘价)

**评分规则**：
```python
位置<15% + 波动率<5%（极低位+缩量）  → 15分 ⭐满分
位置<15% + 波动率≥5%              → 13分
位置<25% + 波动率<8%（低位区域）   → 12分
位置<25% + 波动率≥8%              → 10分
位置<35%（相对低位）               → 8分
位置<45%                         → 5分
位置<60%                         → 2分
```

**代码位置**：`_score_bottom_feature()` 函数（第1865-1892行）

---

### 🏦 维度5：主力吸筹（10分）

**计算逻辑**：
- 量比 = 近3天均量 / 历史20天均量
- 价格稳定天数 = 近5天中涨跌幅≥-1%的天数
- 连续放量天数 = 近5天中成交量>均量1.2倍的天数

**评分规则**：
```python
温和放量(1.3≤量比≤1.8) + 小涨(0%≤涨幅≤3%) 
  + 稳定(≥4天)                    → 10分 ⭐满分
温和放量(1.3≤量比≤1.8) + 小涨     → 7分
连续放量(≥3天) + 不跌             → 8分
放量(1.5≤量比≤2.0) + 上涨         → 6分
温和放量(量比>1.2) + 上涨         → 4分
```

**代码位置**：`_score_main_force_accumulation()` 函数（第1894-1915行）

---

### ⚡ 维度6：涨停基因（5分）

**计算逻辑**：
- 涨停 = 当日涨幅 > 9.5%
- 统计近5天和近20天涨停次数

**评分规则**：
```python
近5天涨停≥2次      → 5分 ⭐满分
近5天涨停≥1次      → 3分
近20天涨停≥2次     → 2分
近20天涨停≥1次     → 1分
无涨停记录         → 0分
```

**代码位置**：`_score_limit_up_gene()` 函数（第1917-1935行）

## ⚠️ AI优化3：风险扣分（0-41分）

当股票存在系统性风险时，即使基础分高也要扣分降级。

### ⚠️ 风险1：高位回落风险（-0~10分）

**扣分规则**：
```python
60天涨幅>50% + 近5天下跌>3%   → -10分（高位出货）
60天涨幅>40% + 近5天下跌>3%   → -7分
60天涨幅>30% + 价格位置>70%   → -5分
```

---

### 📊 风险2：波动率风险（-0~8分）

**扣分规则**：
```python
波动率 > 15%  → -8分（剧烈震荡）
波动率 > 12%  → -6分
波动率 > 10%  → -4分
```

*波动率 = std(近20日) / mean(近20日)*

---

### 💥 风险3：暴跌风险（-0~8分）

**扣分规则**：
```python
近60天跌停≥3次  → -8分（频繁暴跌）
近60天跌停≥2次  → -6分
近60天跌停≥1次  → -3分
```

*跌停 = 单日跌幅 < -9.5%*

---

### 📉 风险4：技术破位风险（-0~10分）

**扣分规则**：
```python
当前价 < MA5 < MA10 < MA20  → -10分（完全空头）
当前价 < MA20               → -6分（跌破中期均线）
当前价 < MA10               → -3分（跌破短期均线）
```

---

### 💧 风险5：流动性风险（-0~5分）

**扣分规则**：
```python
量比 < 0.5  → -5分（严重缩量）
量比 < 0.7  → -3分（缩量）
```

**代码位置**：`_calculate_risk_penalty()` 函数（第1999-2045行）

## 📐 完整计算示例

### 假设股票A的技术数据

**基础数据**：
- 当前价：10.50元
- 60日最高：12.00元，60日最低：8.00元
- 近3天均量：500万手，历史20日均量：300万手
- 近5天涨幅：+4.5%

**均线数据**：
- MA5：10.20元，MA10：9.80元，MA20：9.50元，MA60：9.20元
- MACD：DIF=0.05, DEA=0.02，DIF和DEA都在上升

**其他数据**：
- 近5天价格稳定天数：4天
- 近20日波动率：6%
- 近5天涨停次数：1次
- 近60天跌停次数：0次

---

### 第一步：计算6个基础维度得分

**维度1：量价配合**
```
量比 = 500 / 300 = 1.67
涨幅 = +4.5%
价格位置 = (10.5 - 8.0) / (12.0 - 8.0) = 62.5%

→ 量比>1.5 且 涨幅>2% 且 位置<70%
→ 得分：20分
```

**维度2：均线多头**
```
MA5(10.20) > MA10(9.80) > MA20(9.50) > 0
涨幅 = +4.5% > 2%

→ 完美多头排列 + 涨幅>2%
→ 得分：20分 ⭐满分
```

**维度3：MACD趋势**
```
DIF(0.05) > DEA(0.02) → 金叉
DIF上升 ✓, DEA上升 ✓, 柱线上升 ✓
DIF > 0（零轴上）

→ 完美三向上 + 零轴上
→ 得分：25分 ⭐满分
```

**维度4：底部特征**
```
价格位置 = 62.5% （较高位置）
波动率 = 6%

→ 位置>60%
→ 得分：2分
```

**维度5：主力吸筹**
```
量比 = 1.67（在1.5~2.0之间）
涨幅 = +4.5% > 0%

→ 放量上涨
→ 得分：6分
```

**维度6：涨停基因**
```
近5天涨停次数 = 1次

→ 得分：3分
```

**基础得分 = 20 + 20 + 25 + 2 + 6 + 3 = 76分**

---

### 第二步：计算协同加分

检查4个黄金组合：

1. **完美启动**（位置<20% + 量比>2.0 + 金叉）
   - 位置=62.5% ❌ → 不满足

2. **主力吸筹**（温和放量 + 稳定 + 低位 + 小涨）
   - 量比1.67在1.3~1.8区间 ✓
   - 稳定天数4天 ✓
   - 但位置62.5%不在低位 ❌ → 不满足

3. **突破确认**（多头排列 + 放量 + 涨停基因）
   - 多头排列 ✓
   - 量比1.67 < 1.8 ❌ → 不满足

4. **底部蓄势完成**（低位 + 缩量 + 放量）
   - 位置62.5%不在低位 ❌ → 不满足

**协同加分 = 0分**

---

### 第三步：计算风险扣分

检查5类风险：

1. **高位回落风险**：60天涨幅 = 14.1% < 30% → 扣0分
2. **波动率风险**：波动率 = 6% < 10% → 扣0分
3. **暴跌风险**：近60天跌停0次 → 扣0分
4. **技术破位风险**：当前价10.5 > MA5 > MA10 > MA20（多头排列）→ 扣0分
5. **流动性风险**：量比 = 1.67 > 0.7 → 扣0分

**风险扣分 = 0分**

---

### 第四步：计算最终得分和评级

```python
最终得分 = 基础得分 + 协同加分 - 风险扣分 + 行业加分
         = 76 + 0 - 0 + 0
         = 76分
```

**评级判定**：76分 → 75-79分区间 → 💡 C级股票

**策略建议**：
- ✅ 优势：均线多头排列完美（20分满分），MACD趋势强劲（25分满分）
- ⚠️ 劣势：价格位置偏高（62.5%），底部特征得分低（2分），无黄金组合加分
- 📊 结论：技术形态良好，但价格位置不够理想，建议等待回调到更低位置再介入
