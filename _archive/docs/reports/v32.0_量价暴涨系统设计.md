# 🚀 v32.0 专业量价暴涨挖掘系统

## 核心设计理念

**目标**：穷尽所有量价关系，精准预测即将暴涨的股票

## 一、深度量价关系分析引擎（8大核心维度）

### 1. 量价背离分析 (0-15分)
```python
- 顶背离识别：价格新高但成交量萎缩 → 警示信号
- 底背离识别：价格新低但成交量萎缩 → 买入信号
- 背离强度计算：连续背离天数、背离幅度
- 评分逻辑：底背离得高分，顶背离扣分
```

### 2. 量价共振分析 (0-15分)
```python
- 价涨量增：最佳形态，15分
- 价跌量缩：次优形态，12分
- 价涨量缩：警惕信号，5分
- 价跌量增：危险信号，0分
- 共振强度：计算相关系数
```

### 3. 主力资金监控 (0-15分)
```python
- 大单流入：统计大于100万的单笔成交
- 主力控盘度：计算主力持仓比例变化
- 资金流向：计算近5日、10日、20日资金净流入
- 机构席位：分析龙虎榜数据
```

### 4. 异常放量识别 (0-10分)
```python
- 放量突破：成交量>均量2倍 + 突破压力位
- 天量识别：成交量>近期最大量1.5倍
- 持续放量：连续3天放量
- 底部放量：低位放量 → 主力建仓信号
```

### 5. 缩量上涨识别 (0-10分)
```python
- 缩量上涨：价格上涨但量能萎缩
- 强势特征：主力控盘，不需要量能推动
- 持续性：连续缩量上涨天数
- 空间预测：基于历史缩量上涨幅度
```

### 6. 突破确认系统 (0-10分)
```python
- 突破类型：
  • 箱体突破
  • 平台突破
  • 颈线突破
  • 均线突破
- 确认条件：
  • 放量突破
  • 站稳3日
  • 回踩不破
```

### 7. 趋势强度评估 (0-10分)
```python
- ADX指标：趋势强度
- 均线系统：多头排列程度
- 波段高低点：连续新高
- 上涨斜率：角度越陡越强
```

### 8. 买卖盘力道分析 (0-15分)
```python
- 内外盘比：主动买入/卖出比例
- 大单比例：大单占比越高越强
- 分时形态：尾盘拉升、早盘冲高等
- 委买委卖：挂单量对比
```

**总计：100分制，全方位量价评估**

## 四、预测评分模型

### 综合评分公式
```python
最终评分 = (
    量价关系分 × 0.40 +        # 40% - 核心
    暴涨DNA匹配度 × 0.30 +      # 30% - 关键
    技术指标分 × 0.15 +         # 15%
    资金流向分 × 0.10 +         # 10%
    市场情绪分 × 0.05           # 5%
)

评级标准：
90-100分：💎 必涨股（强烈推荐）
80-89分：⭐ 优质股（推荐）
70-79分：✅ 潜力股（可关注）
60-69分：⚠️  观察股（谨慎）
<60分：❌ 不推荐
```

### 涨幅预测
```python
基于历史数据回归：
预测涨幅 = f(
    当前评分,
    暴涨DNA类型,
    所在板块,
    市场环境,
    历史相似案例平均涨幅
)

预测周期：
短期（3-5天）
中期（5-15天）
长期（15-30天）
```
