### 方案C：调整v7.0参数（如果问题严重）⚙️

如果诊断发现严重问题，可能需要调整：

#### 1. 放宽三层过滤

**当前标准太严**:
```python
# 过滤器1：市场情绪
if sentiment < -0.7:  # 极度恐慌
    required_score = 80  # 太高了

# 过滤器2：行业景气度
if industry_heat < -0.5:  # 行业很冷
    required_score = 70  # 太高了
```

**建议调整**:
```python
# 过滤器1：市场情绪（放宽）
if sentiment < -0.8:  # 更极端才提高门槛
    required_score = 75  # 降低要求

# 过滤器2：行业景气度（放宽）
if industry_heat < -0.7:  # 更冷才过滤
    required_score = 65  # 降低要求
```

#### 2. 调整行业轮动加分

**当前可能问题**:
```python
# 热门行业加分可能导致追高
if industry in self.hot_industries:
    rank = self.hot_industries.index(industry) + 1
    if rank == 1:
        bonus_score = 10  # 第1名加10分，可能追高
```

**建议调整**:
```python
# 降低热门行业加分，避免追高
if industry in self.hot_industries:
    rank = self.hot_industries.index(industry) + 1
    if rank == 1:
        bonus_score = 5  # 降低加分
    elif rank <= 3:
        bonus_score = 3
```

#### 3. 调整动态权重

如果发现某个市场环境下权重设置不合理，需要微调。
