## 🔍 常见问题

### Q1：v3.0和v2.0的主要区别是什么？

**A**：核心区别是理念转变：
- v2.0：底部为王，强调股票处于底部位置
- v3.0：启动为王，强调底部股票已经启动

具体变化：
1. ⭐ 新增【启动确认】维度（20分）
2. ⭐ 新增【技术指标】维度（10分）
3. ❌ 移除【筹码分布】维度（数据不可靠）
4. 🔄 协同加分从3种扩展到6种黄金组合
5. 🔄 风险扣分从0-50分降低到0-30分

---

### Q2：v3.0是否向后兼容v2.0？

**A**：返回值结构基本兼容，但有以下区别：
- ✅ 核心字段兼容：`comprehensive_score`, `final_score`, `grade`
- ✅ 维度评分字段：`dimension_scores`（但维度名称不同）
- ⚠️ 新增字段：`combo_type`, `risk_reasons`, `description`
- ❌ 移除字段：筹码分布相关字段

**建议**：使用 `result.get()` 方法获取字段，避免KeyError

### 1. 选股流程

```python
# 完整的选股流程示例
def comprehensive_stock_selection_v3(
    market_cap_min=100,  # 最小市值（亿）
    market_cap_max=500,  # 最大市值（亿）
    score_threshold=75,  # 评分阈值（A级）
    max_stocks=20        # 最多选股数
):
    evaluator = ComprehensiveStockEvaluatorV3()
    
    # 1. 获取股票池（市值筛选）
    stock_pool = get_stocks_by_market_cap(market_cap_min, market_cap_max)
    
    # 2. 逐只评分
    scored_stocks = []
    for ts_code in stock_pool:
        stock_data = get_stock_data(ts_code)
        result = evaluator.evaluate_stock_v3(stock_data)
        
        if result['success'] and result['comprehensive_score'] >= score_threshold:
            scored_stocks.append({
                **result,
                'ts_code': ts_code
            })
    
    # 3. 排序：优先S级，然后按评分排序
    scored_stocks.sort(key=lambda x: (
        0 if x['grade'] == 'S' else 1 if x['grade'] == 'A' else 2,
        -x['comprehensive_score']
    ))
    
    # 4. 返回TOP N
    return scored_stocks[:max_stocks]
```

```python
from concurrent.futures import ProcessPoolExecutor

def batch_evaluate_v3(stock_list, num_workers=4):
    """批量评分（多进程）"""
    evaluator = ComprehensiveStockEvaluatorV3()
```

### 2. 自定义权重

```python
# 自定义评分维度权重（高级）
def custom_weighted_score_v3(result, custom_weights):
    """
    自定义权重计算最终得分
    
    custom_weights示例:
    {
        '启动确认': 0.25,  # 默认0.20
        '底部特征': 0.10,  # 默认0.15
        '量价配合': 0.20,  # 默认0.15
        '技术指标': 0.15,  # 默认0.10
        ...
    }
    """
    dimension_scores = result['dimension_scores']
    
    # 归一化权重
    total_weight = sum(custom_weights.values())
    normalized_weights = {k: v/total_weight for k, v in custom_weights.items()}
    
    # 计算加权得分
    weighted_score = sum(
        dimension_scores[dim] * normalized_weights.get(dim, 0)
        for dim in dimension_scores
    )
    
    # 加上协同加分，减去风险扣分
    final_score = weighted_score + result['synergy_bonus'] - result['risk_penalty']
    final_score = max(0, min(100, final_score))
    
    return final_score
```
