# v8.0 系统使用指南

## 启动

```bash
python3 start_v8_enhanced_system.py
```

指定端口：

```bash
python3 start_v8_enhanced_system.py --port 8502
```

## 直接启动（备用）

```bash
streamlit run 机构级V13系统_v6.1_最完美融合版.py --server.port 8501
```

## 快速验证

```bash
python3 test_v8_scan_validation.py
python3 test_v8_integration.py
```

## 依赖

- Python 3.10+
- streamlit
- pandas
- numpy
- plotly
