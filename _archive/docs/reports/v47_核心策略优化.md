### 2. 成交量质量评分（关键创新）

```python
# 传统方法（v46）
volume_surge = 当前量 / 均量
→ 问题：无法区分真假放量

# v47方法
1. 计算OBV（能量潮）
2. 检查OBV趋势（5日变化）
3. 综合评分：
   - 放量2倍 + OBV上升10% = 20分（真放量）
   - 放量2倍 + OBV下降 = 5分（假放量）
```

### 3. 趋势强度量化（ADX简化版）

```python
trend_strength = abs(价格变化) / 价格波动率 / 周期
- >0.5: 强趋势（优先选择）
- 0.3-0.5: 中等趋势（可接受）
- <0.3: 弱趋势（淘汰）

作用：避免震荡市产生大量假信号
```

### 4. 动态可靠度计算

```python
reliability = 基础分(0.4) + 评分贡献(score/150) + 确认贡献(confirmations*0.05)

示例：
- 评分80分 + 4个确认 = 0.4 + 0.53 + 0.20 = 1.13 → 限制到1.0
- 评分75分 + 3个确认 = 0.4 + 0.50 + 0.15 = 1.05 → 限制到1.0  
- 评分70分 + 2个确认 = 0.4 + 0.47 + 0.10 = 0.97 ✅
- 评分70分 + 1个确认 = 0.4 + 0.47 + 0.05 = 0.92 ✅
- 评分60分 + 2个确认 = 0.4 + 0.40 + 0.10 = 0.90 ✅
- 评分60分 + 1个确认 = 0.4 + 0.40 + 0.05 = 0.85 ✅
- 评分50分 + 2个确认 = 0.4 + 0.33 + 0.10 = 0.83 ✅
```
