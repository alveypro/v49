### ✅ 优化4：统一市值单位转换

**优化内容**:
创建统一的市值转换工具（`utils_database.py`）：

```python
def wan_to_yi(value_wan: float) -> float:
    """万元转换为亿元"""
    return value_wan / 10000

def yi_to_wan(value_yi: float) -> float:
    """亿元转换为万元"""
    return value_yi * 10000

def format_market_cap(value_wan: float, precision: int = 2) -> str:
    """格式化市值显示"""
    value_yi = wan_to_yi(value_wan)
    return f"{value_yi:.{precision}f}亿"
```
