## 📊 核心功能详解

### 1️⃣ 每日选股

**功能**: 基于v4.0策略自动筛选高分股票

**使用方法**:
```python
from trading_assistant import TradingAssistant
assistant = TradingAssistant()

# 执行选股
recommendations = assistant.daily_stock_scan(top_n=5)

# 查看结果
for rec in recommendations:
    print(f"{rec['stock_name']}: {rec['score']:.1f}分 @ ¥{rec['price']:.2f}")
```

**在Streamlit中**:
1. 进入"智能交易助手" Tab
2. 点击"每日选股"子页
3. 设置推荐数量
4. 点击"开始选股"
5. 等待2-3分钟
6. 查看推荐结果

**推荐参数**:
- 最低评分: 65分
- 市值范围: 100-500亿
- 推荐数量: 5只

---

### 2️⃣ 持仓管理

**功能**: 记录和跟踪您的买卖操作

**添加持仓**（买入后记录）:
```python
assistant.add_holding(
    ts_code='600519.SH',      # 股票代码
    buy_price=1800.00,         # 买入价格
    quantity=100,              # 数量
    score=78.5                 # 评分（可选）
)
```

**更新持仓**（获取最新价格）:
```python
assistant.update_holdings()
```

**卖出持仓**:
```python
assistant.sell_holding(
    ts_code='600519.SH',
    sell_price=1900.00,
    reason='止盈6%'
)
```

**在Streamlit中**:
1. 进入"持仓管理"子页
2. 点击"更新持仓"按钮
3. 查看各持仓盈亏情况
4. 需要卖出时填写卖出价格和原因
5. 点击"卖出"按钮

---

### 3️⃣ 止盈止损监控

**功能**: 自动检测并提醒止盈止损

**检查止盈止损**:
```python
alerts = assistant.check_stop_conditions()

for alert in alerts:
    print(alert['message'])
    # 例如: "🎉 贵州茅台 已达止盈条件！盈利6.67%，建议卖出！"
```

**默认参数**:
- 止盈: +6%
- 止损: -4%

**在Streamlit中**:
- 进入"持仓管理"
- 系统自动显示提醒
- 红色=止损，绿色=止盈

### 5️⃣ 定时任务

**功能**: 自动化执行日常任务

**启动定时服务**:
```bash
python scheduler_service.py
```

**定时任务列表**:

| 任务 | 时间 | 说明 |
|------|------|------|
| 每日选股 | 9:15 | 自动筛选推荐股票 |
| 持仓监控 | 10:30, 14:00, 15:00 | 检查止盈止损 |
| 每日报告 | 18:00 | 生成并发送报告 |

**后台运行**（Linux/Mac）:
```bash
nohup python scheduler_service.py > scheduler.log 2>&1 &
```

**查看日志**:
```bash
tail -f scheduler.log
```

**停止服务**:
```bash
ps aux | grep scheduler_service
kill [PID]
```

## 🔧 常见问题

### Q1: 选股太慢怎么办？
**A**: 选股需要扫描和评分，正常需要2-3分钟。可以：
- 减少推荐数量（3只即可）
- 使用定时任务（后台自动运行）
- 提高最低评分（减少候选数量）

### Q2: 没有收到通知？
**A**: 检查：
1. notification_config.json中enabled是否为true
2. 邮箱授权码是否正确（不是登录密码）
3. 查看日志：tail -f scheduler.log

### Q3: 定时任务没有执行？
**A**: 
1. 检查scheduler_service.py是否在运行：`ps aux | grep scheduler`
2. 查看日志：`cat scheduler.log`
3. 确认当前时间是否到了执行时间

### Q4: 如何修改定时任务时间？
**A**: 编辑 scheduler_service.py:
```python
# 找到这行
CronTrigger(hour=9, minute=15)
# 修改为您想要的时间
```

### Q5: 可以自动下单吗？
**A**: 不能！这是设计理念：
- 系统只提供建议
- 需要您人工审核
- 您人工下单
- 系统记录和监控
- 达到条件提醒

这样更安全、可控！
