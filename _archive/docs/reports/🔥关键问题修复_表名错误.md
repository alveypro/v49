# 🔥 关键问题修复：表名错误导致评分失败

**发现时间**: 2024-12-24  
**问题严重性**: ⚠️ 致命错误  
**影响范围**: v4.0策略完全无法评分

---

## ❌ 问题描述

### 现象
- ✅ 能找到符合市值条件的股票
- ❌ 但评分后没有任何结果
- ❌ 即使降低评分阈值也没用

### 根本原因
**第4593行代码使用了错误的表名**：

```python
# ❌ 错误的表名
data_query = """
    SELECT trade_date, close_price as close, vol as volume, pct_chg
    FROM daily_data          # ❌ 这个表不存在！
    WHERE ts_code = ?
    ORDER BY trade_date DESC
    LIMIT 120
"""
```

**正确的表名应该是**: `daily_trading_data`

### 导致的后果
1. 查询历史数据时失败
2. `stock_data` 为空
3. 无法进行评分
4. 所有股票都被跳过
5. 最终结果为空

---

## ✅ 修复方案

### 修复代码

**位置**: 第4593行

**修改前**:
```python
data_query = """
    SELECT trade_date, close_price as close, vol as volume, pct_chg
    FROM daily_data                    # ❌ 错误
    WHERE ts_code = ?
    ORDER BY trade_date DESC
    LIMIT 120
"""
```

**修改后**:
```python
data_query = """
    SELECT trade_date, close_price as close, vol as volume, pct_chg
    FROM daily_trading_data            # ✅ 正确
    WHERE ts_code = ?
    ORDER BY trade_date DESC
    LIMIT 120
"""
```
