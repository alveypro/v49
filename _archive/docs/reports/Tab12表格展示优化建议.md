### 2. **筹码分布**列 🔥 新增
```python
# 建议显示格式

示例：
"18分 (28%)" - 优秀，低获利

"5分 (72%)" - 警惕，高获利

# 颜色编码
获利比例 < 30%: 绿色（安全）
获利比例 30-50%: 黄色（谨慎）
获利比例 > 50%: 红色（危险）
```

### 3. **价格位置**列
```python
# 建议显示格式

"15% 极低位" - 绿色

"50% 中位" - 黄色
"75% 高位" - 橙色

# 颜色编码
< 20%: 深绿色（最佳买点）
20-40%: 浅绿色（买点）
40-60%: 黄色（观望）
60-80%: 橙色（谨慎）
> 80%: 红色（风险）
```

### 4. **放量倍数**列
```python
# 建议显示格式

## 📊 表格排序建议

```

### 可选排序选项：

3. **底部特征** ↓（找底部股）

## 📊 数据格式化建议

### 数值类
```python
得分：整数或一位小数 "85.5"
价格：两位小数 "¥23.45"
市值：一位小数 + 单位 "156.8亿"
百分比：一位小数 + % "25.3%"
倍数：两位小数 + 倍 "1.85倍"
```

### 文本类
```python
等级：表情 + 文字 "💎 S级"
筹码状态：简短描述 "低位 | 放量 | 上涨"
行业：简称 "新能源" 而非 "电气设备-新能源设备"
```

### 空值处理
```python
有数据：正常显示
无数据：显示 "-" 或 "N/A"
不显示空白或null
```
