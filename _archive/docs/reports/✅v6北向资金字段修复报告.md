# ✅ v6北向资金字段修复报告

**修复日期**: 2026-01-07 08:20  
**问题**: v6回测时大量警告 `'north_net_3d'` 字段缺失  
**原因**: 数据提供器返回字段名与评分器期望字段名不匹配  
**状态**: ✅ 已修复

---

## 🔍 问题描述

### 错误日志

```
WARNING - 资金流向评分失败 002506.SZ: 'north_net_3d'
WARNING - 资金流向评分失败 002506.SZ: 'north_net_3d'
... (数百条相同警告)
```

### 影响

- v6回测时无法正确评估资金流向维度
- 评分不准确
- 大量警告日志影响性能

---

## 🐛 根本原因

### 字段名不匹配

**数据提供器返回**（v6_data_provider.py）:
```python
{
    'buy_amount': xxx,
    'sell_amount': xxx,
    'net_amount': xxx,  # ← 返回这个字段
    'consecutive_buy_days': xxx,
    'is_connect_stock': True
}
```

**v6评分器期望**（comprehensive_stock_evaluator_v6_ultimate.py）:
```python
north_net = north_flow['north_net_3d']  # ← 期望这个字段
```

### 问题定位

- 第390行：v6评分器尝试访问 `north_flow['north_net_3d']`
- 数据提供器只返回 `net_amount`，没有 `north_net_3d`
- 导致KeyError，触发异常警告

---

## 🛠️ 修复方案

### 方案：添加字段别名

在数据提供器的所有返回值中，添加 `north_net_3d` 作为 `net_amount` 的别名。

### 修复文件

1. `v6_data_provider.py`（标准版）
2. `v6_data_provider_optimized.py`（优化版）
