## 🔍 问题根源分析

### 问题1：追高风险 ⚠️⚠️⚠️

**v5.0的核心理念**："启动确认"后买入
- 要求股票已经站上均线
- 要求连续阳线
- 要求放量上涨

**问题**：当这些条件都满足时，股票往往已经涨了一波，买入点偏高，容易回调。

**数据支持**：
- 42.2%的交易触发止损（-3%）
- 平均持仓只有3.3天就止损，说明买入后快速下跌

### 问题2：评分标准与实际收益不匹配 ❌

**数据显示**：
- 60-65分：胜率40.8%，收益+0.01%
- 65-70分：胜率38.1%，收益-0.49% ❌
- 70-75分：胜率39.4%，收益+0.75%
- 75+分：胜率35.6%，收益+0.53%

**结论**：评分越高，表现反而越差！这说明v5.0的评分逻辑有问题。

### 问题3：止盈比例设置不当 ⚠️

**数据**：
- 止盈（+4%）：385次，平均收益+7.39%
- 止损（-3%）：645次，平均收益-4.39%

**问题**：
- 止盈次数太少（25.2%），说明+4%的目标太高
- 止损次数太多（42.2%），说明买入时机不对

### 问题4：市场环境不适合"启动确认"策略 📉

**当前市场特征**：
- 大盘震荡下跌
- 个股普遍走弱
- "启动"后往往是假突破

**v5.0策略特点**：
- 追求"确认趋势后买入"
- 适合牛市或强势市场
- 不适合震荡或熊市

## 💡 优化方案

### 方案A：优化v5.0评分标准（深度优化）

#### 1. 降低"启动确认"的权重

**当前**：启动确认20分（最高权重）

**优化**：
- 启动确认降低到15分
- 增加"安全边际"维度（5分）
  - 价格位置<50%：+5分
  - 价格位置<30%：+3分
  - 价格位置>70%：-5分（高位风险）

#### 2. 增加"回调买入"逻辑

**新增评分维度**：
- 短期回调（5分）：
  - 3天内回调3-5%：+5分（回调买入机会）
  - 3天内回调5-8%：+3分
  - 3天内上涨>5%：-3分（追高风险）

#### 3. 优化风险扣分

**当前问题**：风险扣分不够严格

**优化**：
- 高位风险（价格位置>70%）：-15分（加重）
- 短期暴涨（3天涨>10%）：-10分（新增）
- 成交量异常放大（量比>5）：-8分（新增）

#### 4. 调整协同加分

**当前问题**：协同加分鼓励"已启动"状态

**优化**：
- 删除"强势启动"加分（启动+主力）
- 新增"安全潜伏"加分：
  - 底部（<30%）+ 温和放量 + MACD金叉：+8分
  - 回调（-3~-5%）+ 主力流入：+6分

### 方案B：创建v5.1"回调买入版"（推荐）⭐⭐⭐

**核心理念改变**：
- v5.0：启动确认后买入（追高）
- v5.1：启动后回调时买入（低吸）

**新的8维评分体系**：

| 维度 | v5.0分值 | v5.1分值 | 调整理由 |
|------|---------|---------|---------|
| 潜伏价值 | 10分 | 15分 | ⬆️ 提高，重视底部 |
| 底部特征 | 10分 | 15分 | ⬆️ 提高，重视底部 |
| 量价配合 | 20分 | 15分 | ⬇️ 降低，不追求放量 |
| MACD趋势 | 7分 | 10分 | ⬆️ 提高，重视趋势 |
| 均线多头 | 7分 | 10分 | ⬆️ 提高，重视趋势 |
| 主力行为 | 18分 | 15分 | ⬇️ 降低，不强求放量 |
| **回调买入** | 0分 | **15分** | 🆕 新增核心维度 |
| 涨停基因 | 8分 | 5分 | ⬇️ 降低，不重要 |

**回调买入评分逻辑**（15分）：
```python
def _score_pullback_entry(self, ind: Dict, close, pct_chg) -> float:
    """
    【核心维度】回调买入（15分）
    
    在趋势确立后，等待回调时买入
    """
    score = 0
    
    # 1. 趋势确立（必要条件）
    ma20 = ind['ma20']
    ma60 = ind['ma60']
    current_price = close[-1]
    
    trend_confirmed = current_price > ma60  # 站上60日线
    
    if not trend_confirmed:
        return 0  # 趋势未确立，不考虑
    
    # 2. 短期回调（核心）
    price_chg_3d = sum(pct_chg[-3:]) / 100  # 近3天涨跌幅
    
    if -0.08 <= price_chg_3d <= -0.03:
        score += 15  # 回调3-8%，最佳买点！
    elif -0.10 <= price_chg_3d < -0.08:
        score += 10  # 回调8-10%，较好买点
    elif -0.03 < price_chg_3d <= 0:
        score += 8   # 轻微回调，可以考虑
    elif price_chg_3d > 0.05:
        score -= 5   # 短期上涨>5%，追高风险
    
    return score
```
