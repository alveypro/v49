## 📈 评分逻辑

### 完整计算流程

```
步骤1：计算6维基础得分（0-100分）
├─ 量价配合：0-25分
├─ 均线多头：0-20分
├─ MACD趋势：0-25分
├─ 底部特征：0-15分
├─ 主力吸筹：0-10分
└─ 涨停基因：0-5分

步骤2：协同效应加分（0-20分）
├─ 完美启动组合：+10分
├─ 主力吸筹组合：+8分
├─ 突破确认组合：+8分
└─ 底部蓄势组合：+6分

步骤3：风险评分扣分（0-41分）
├─ 高位风险：-10分
├─ 波动风险：-8分
├─ 暴跌风险：-8分
├─ 技术破位：-10分
└─ 流动性风险：-5分

步骤4：行业热度加分（0-5分）

步骤5：计算最终得分
最终得分 = 基础得分 + 协同加分 - 风险扣分 + 行业加分
最终得分 = max(0, min(100, 最终得分))

步骤6：评级
S级：≥90分（顶级）
A级：85-89分（优质）
B级：80-84分（良好）
C级：75-79分（合格）
D级：70-74分（一般）
E级：<70分（不推荐）
```

## 🔍 常见问题

### Q1：为什么高分股票很少？

**A**: 这是正常的！新评分体系更严格、更科学。

- S级（≥90分）：<0.1%（极其罕见）
- A级（85-89分）：1-2%（优质股）
- B级（80-84分）：3-5%（良好股）

**这正是优化的目标**：只推荐真正优质的股票！

### Q2：为什么有些股票协同分是0？

**A**: 协同加分需要满足特定的黄金组合条件：

- 完美启动：低位(<20%) + 强势放量(>2.0) + MACD金叉
- 主力吸筹：温和放量(1.3-1.8) + 稳定 + 低位(<35%)

如果不满足任何组合，协同分就是0。

### Q3：建议止损价是如何计算的？

**A**: 智能选择3种方法中最保守的：

1. ATR动态止损：当前价 - 2×ATR
2. MA20支撑止损：MA20 × 0.95
3. 8%固定止损：当前价 × 0.92

取三者中最高的（最保守的）作为建议止损价。

## 📚 技术文档

### 核心函数

```python
# 主评分函数
evaluate_stock_comprehensive(stock_data) -> Dict

# 6维评分函数
_score_volume_price()           # 维度1：量价配合
_score_ma_trend()               # 维度2：均线多头
_score_macd_trend()             # 维度3：MACD趋势
_score_bottom_feature()         # 维度4：底部特征
_score_main_force_accumulation() # 维度5：主力吸筹
_score_limit_up_gene()          # 维度6：涨停基因

# AI优化函数
_detect_market_environment()    # 市场环境识别
_detect_stock_stage()           # 个股阶段识别
_calculate_synergy_bonus()      # 协同效应加分
_calculate_risk_penalty()       # 风险评分扣分
_recommend_stop_loss()          # 智能止损推荐
```

### 返回值结构

```python
{
    'comprehensive_score': float,  # 综合得分（0-100）

        'MACD趋势': 0-25,
        '底部特征': 0-15,
        '主力吸筹': 0-10,

    'grade': str,                 # 评级（S/A/B/C/D/E）
    'market_env': str,            # 市场环境
    'stock_stage': str,           # 个股阶段
    'stop_loss': float,           # 建议止损价
    'stop_loss_method': str,      # 止损方法
```
