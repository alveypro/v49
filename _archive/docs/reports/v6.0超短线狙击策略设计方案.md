### 维度1：板块热度（25分）⭐⭐⭐

**核心理念**：中国A股板块联动效应极强，选对板块比选对股票更重要！

```python
def _score_sector_heat(self, stock_code, sector_name) -> float:
    """
    板块热度评分（25分）
    
    评分标准：
    1. 板块近3日涨幅（10分）
       - 板块涨幅>5%：10分 ⭐⭐⭐
       - 板块涨幅3-5%：7分
       - 板块涨幅1-3%：4分
       - 板块涨幅<1%：0分
    
    2. 板块资金流入（10分）
       - 板块大单净流入>10亿：10分 ⭐⭐⭐
       - 板块大单净流入5-10亿：7分
       - 板块大单净流入1-5亿：4分
       - 板块资金净流出：0分
    
    3. 板块题材热度（5分）
       - 政策利好/热门题材：5分 ⭐
       - 一般题材：2分
       - 无题材：0分
    """
    score = 0
    
    # 1. 获取板块数据
    sector_data = get_sector_data(sector_name, days=3)
    
    # 2. 板块涨幅
    sector_change = sector_data['change_3d']
    if sector_change > 5:
        score += 10
    elif sector_change > 3:
        score += 7
    elif sector_change > 1:
        score += 4
    
    # 3. 板块资金流入
    sector_money_flow = sector_data['money_flow_3d']  # 亿元
    if sector_money_flow > 10:
        score += 10
    elif sector_money_flow > 5:
        score += 7
    elif sector_money_flow > 1:
        score += 4
    
    # 4. 题材热度
    if sector_data['is_hot_topic']:
        score += 5
    elif sector_data['has_policy_support']:
        score += 3
    
    return min(score, 25)
```

**数据来源**：
- 板块涨跌幅：Tushare板块指数
- 板块资金流入：同花顺/东方财富板块资金流
- 题材热度：人工标注/新闻热度

**核心理念**：跟随聪明钱，资金流向是最真实的信号！

```python
def _score_money_flow(self, stock_data, stock_code) -> float:
    """
    资金流向评分（20分）
    
    评分标准：
    1. 大单净流入（10分）
       - 连续3天大单净流入：10分 ⭐⭐⭐
       - 今日大单净流入>5000万：7分
       - 今日大单净流入>1000万：4分
       - 大单净流出：0分
    
    2. 主力资金（5分）
       - 主力资金连续流入：5分 ⭐
       - 主力资金今日流入：3分
       - 主力资金流出：0分
    
    3. 北向资金（5分）- 如果是沪深港通标的
       - 北向资金连续买入：5分 ⭐
       - 北向资金今日买入：3分
       - 无北向数据：0分
    """
    score = 0
    
    # 1. 大单净流入
    large_order_flow = get_large_order_flow(stock_code, days=3)
    if large_order_flow['consecutive_inflow_days'] >= 3:
        score += 10
    elif large_order_flow['today'] > 5000:  # 万元
        score += 7
    elif large_order_flow['today'] > 1000:
        score += 4
    
    # 2. 主力资金
    main_force_flow = get_main_force_flow(stock_code, days=3)
    if main_force_flow['consecutive_inflow_days'] >= 3:
        score += 5
    elif main_force_flow['today'] > 0:
        score += 3
    
    # 3. 北向资金（如果适用）
    if is_connect_stock(stock_code):
        north_flow = get_north_money_flow(stock_code, days=3)
        if north_flow['consecutive_buy_days'] >= 3:
            score += 5
        elif north_flow['today'] > 0:
            score += 3
    
    return min(score, 20)
```

```python
def _score_technical_breakthrough(self, stock_data) -> float:
    """
    技术突破评分（20分）
    
    评分标准：
    1. 突破关键位（10分）
       - 突破平台+放量：10分 ⭐⭐⭐
       - 突破20日线+放量：7分
       - 站上60日线：4分
       - 无明显突破：0分
    
    2. 放量确认（5分）
       - 量比>2.0：5分 ⭐
       - 量比>1.5：3分
       - 量比<1.5：0分
    
    3. 形态完美（5分）
       - 圆弧底/杯柄形态：5分 ⭐
       - 上升三角形：3分
       - 一般形态：1分
    """
    score = 0
    
    # 1. 突破判断
    if is_platform_breakthrough(stock_data, volume_confirm=True):
        score += 10
    elif is_ma20_breakthrough(stock_data, volume_confirm=True):
        score += 7
    elif stock_data['close'][-1] > stock_data['ma60'][-1]:
        score += 4
    
    # 2. 量比确认
    vol_ratio = stock_data['vol_ratio']
    if vol_ratio > 2.0:
        score += 5
    elif vol_ratio > 1.5:
        score += 3
    
    # 3. 形态识别
    pattern = identify_pattern(stock_data)
    if pattern in ['cup_handle', 'rounding_bottom']:
        score += 5
    elif pattern == 'ascending_triangle':
        score += 3
    elif pattern is not None:
        score += 1
    
    return min(score, 20)
```

### 维度4：短期动量（15分）⭐⭐

**核心理念**：超短线要抓强势股，动量越强越好！

```python
def _score_short_term_momentum(self, stock_data) -> float:
    """
    短期动量评分（15分）
    
    评分标准：
    1. 近3天涨幅（10分）
       - 3天涨幅>10%：10分 ⭐⭐
       - 3天涨幅5-10%：7分
       - 3天涨幅3-5%：4分
       - 3天涨幅<3%：0分
    
    2. 连续阳线（5分）
       - 连续3个阳线：5分 ⭐
       - 连续2个阳线：3分
       - 其他：0分
    """
    score = 0
    
    # 1. 近3天涨幅
    change_3d = sum(stock_data['pct_chg'][-3:])
    if change_3d > 10:
        score += 10
    elif change_3d > 5:
        score += 7
    elif change_3d > 3:
        score += 4
    
    # 2. 连续阳线
    positive_days = sum(1 for x in stock_data['pct_chg'][-3:] if x > 0)
    if positive_days == 3:
        score += 5
    elif positive_days == 2:
        score += 3
    
    return min(score, 15)
```

```python
def _score_relative_strength(self, stock_data, stock_code, sector_name) -> float:
    """
    相对强度评分（10分）
    
    评分标准：
    1. 强于大盘（5分）
       - 3天涨幅超大盘>5%：5分 ⭐
       - 3天涨幅超大盘>3%：3分
       - 3天涨幅超大盘>0%：1分
    
    2. 强于板块（5分）
       - 板块排名前3：5分 ⭐
       - 板块排名前10：3分
       - 板块排名前30%：1分
    """
    score = 0
    
    # 1. 相对大盘强度
    stock_change = sum(stock_data['pct_chg'][-3:])
    market_change = get_market_change(days=3)
    relative_strength = stock_change - market_change
    
    if relative_strength > 5:
        score += 5
    elif relative_strength > 3:
        score += 3
    elif relative_strength > 0:
        score += 1
    
    # 2. 板块排名
    sector_rank = get_stock_rank_in_sector(stock_code, sector_name)
    if sector_rank <= 3:
        score += 5
    elif sector_rank <= 10:
        score += 3
    elif sector_rank <= 30:  # 前30%
        score += 1
    
    return min(score, 10)
```

```python
def _calculate_synergy_bonus(self, dim_scores) -> int:
    """
    协同加分（0-15分）
    
    组合1：板块龙头（最强！）
    - 板块热度>20 + 资金流向>15 + 相对强度>8 → +10分 ⭐⭐⭐

    """

    # 组合1：板块龙头
    if (dim_scores['板块热度'] > 20 and

        bonus += 10

    if (dim_scores['板块热度'] > 15 and 
        dim_scores['资金流向'] > 15):
        bonus += 5
    
    return min(bonus, 15)
```

## ⚠️ 风险扣分系统（0-30分）

### 严格风控

```python
def _calculate_risk_penalty(self, stock_data, indicators) -> int:
    """
    风险扣分（0-30分）
    
    1. 追高风险（-15分）
       - 价格位置>80%：-15分 ❌
       - 价格位置>70%：-10分
       - 价格位置>60%：-5分
    
    2. 短期暴涨风险（-10分）
       - 3天涨幅>20%：-10分 ❌
       - 3天涨幅>15%：-5分
    
    3. 其他风险
       - ST股票：-30分（直接淘汰）
       - 停牌风险：-20分
       - 高波动：-5分
    """
    penalty = 0
    
    # 1. 追高风险
    price_pos = indicators['price_position']
    if price_pos > 0.80:
        penalty += 15
    elif price_pos > 0.70:
        penalty += 10
    elif price_pos > 0.60:
        penalty += 5
    
    # 2. 短期暴涨
    change_3d = sum(stock_data['pct_chg'][-3:])
    if change_3d > 20:
        penalty += 10
    elif change_3d > 15:
        penalty += 5
    
    # 3. ST股票（直接淘汰）
    if 'ST' in stock_data['name']:
        penalty += 30
    
    return min(penalty, 30)
```

## 🎯 交易策略

### 买入信号

**条件（必须同时满足）：**
1. 综合评分 ≥ 75分
2. 板块热度 ≥ 15分
3. 资金流向 ≥ 12分
4. 技术突破 ≥ 12分

### 持仓管理

- **持仓周期**：3-5天（超短线）
- **止损**：-5%（严格执行）
- **止盈**：+8%（首次目标），+12%（极限目标）
- **仓位**：单只15-20%，最多5只

### 退出策略

1. **止盈退出**：达到+8%立即止盈50%，剩余50%持有到+12%
2. **止损退出**：跌破-5%立即全部止损
3. **时间退出**：持仓5天无论盈亏全部退出
4. **板块退出**：如果板块转弱（板块3天跌幅>3%），立即退出
