#### 方式B：手动配置

##### 1. QQ邮箱配置（最简单）

**获取授权码**：
1. 登录 [mail.qq.com](https://mail.qq.com)
2. 设置 → 账户 → POP3/IMAP/SMTP服务
3. 开启服务 → 生成授权码
4. 保存16位授权码（如：`abcdefghijklmnop`）

**创建配置文件**：
```bash
cd /Users/mac/QLIB
cat > notification_config.json << 'EOF'
{
  "enabled": true,
  "email": {
    "enabled": true,
    "smtp_server": "smtp.qq.com",
    "smtp_port": 587,
    "sender_email": "你的QQ邮箱@qq.com",
    "sender_password": "你的16位授权码",
    "receiver_emails": ["你的QQ邮箱@qq.com"]
  },
  "serverchan": {
    "enabled": false,
    "sendkey": ""
  },
  "bark": {
    "enabled": false,
    "device_key": ""
  },
  "wechat_work": {
    "enabled": false,
    "webhook_url": ""
  },
  "dingtalk": {
    "enabled": false,
    "webhook_url": "",
    "secret": ""
  }
}
EOF
```

**测试配置**：
```bash
python -c "from notification_service import NotificationService; NotificationService().test_notification()"
```

如果收到邮件，说明配置成功！✅

---

## ❓ 常见问题

### Q1: 我需要一直开着电脑吗？

**A**: 
- 如果要每天自动推送，需要保持运行
- 或者使用云服务器（阿里云/腾讯云）24小时运行
- 或者每天手动启动系统查看推荐

---

### Q2: 系统会自动买卖股票吗？

**A**: 
- **不会！**系统只提供推荐和提醒
- 所有买卖决策由您人工完成
- 这样更安全，您有完全控制权

---

### Q3: 推荐的股票一定赚钱吗？

**A**: 
- **不一定！**没有100%赚钱的策略
- 但根据回测，期望值为正（+0.5-1.0%）
- 长期来看有利可图（年化15-20%）
- 建议小资金测试3-6个月

---

### Q4: 可以修改配置吗？

**A**: 
- 完全可以！在「智能交易助手」Tab中修改
- 可调整：最低评分、止盈止损比例等
- 建议先用默认配置，积累数据后再优化

---

### Q5: 通知发送失败怎么办？

**A**: 
1. 检查配置文件 `notification_config.json`
2. 测试通知：`python 配置通知服务.py`
3. 查看日志：`tail -f scheduler.log`
4. 参考：`📧通知服务完整配置指南.md`
