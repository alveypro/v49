#### 维度4：底部特征（15分）- `_score_bottom_feature()`

#### 维度6：涨停基因（5分）- `_score_limit_up_gene()`

## 📊 评分计算逻辑

### 完整流程

```
步骤1：计算6维基础得分（0-100分）
├─ 量价配合：0-25分
├─ 均线多头：0-20分
├─ MACD趋势：0-25分
├─ 底部特征：0-15分
├─ 主力吸筹：0-10分
└─ 涨停基因：0-5分

步骤2：协同效应加分（0-20分）
├─ 完美启动组合：+10分
├─ 主力吸筹组合：+8分
├─ 突破确认组合：+8分
└─ 底部蓄势组合：+6分

步骤3：风险评分扣分（0-41分）
├─ 高位风险：-10分
├─ 波动风险：-8分
├─ 暴跌风险：-8分
├─ 技术破位：-10分
└─ 流动性风险：-5分

步骤4：行业热度加分（0-5分）

步骤5：计算最终得分
最终得分 = 基础得分 + 协同加分 - 风险扣分 + 行业加分
最终得分 = max(0, min(100, 最终得分))

步骤6：评级
S级：≥90分（顶级）
A级：85-89分（优质）
B级：80-84分（良好）
C级：75-79分（合格）
D级：70-74分（一般）
E级：<70分（不推荐）
```
