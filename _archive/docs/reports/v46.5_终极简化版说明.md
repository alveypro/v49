## 💎 新评分系统（超级简单）

### **基础分：50分** （所有股票都有）

### **维度1：价格涨跌（最多+20分或-10分）**

```python
5天涨幅 > 5%  → +20分  ⭐⭐⭐
5天涨幅 > 3%  → +15分  ⭐⭐
5天涨幅 > 1%  → +10分  ⭐
5天涨幅 > 0%  → +5分
5天跌幅 > -3% → 0分（小跌不扣分）
5天跌幅 < -3% → -10分
```

### **维度2：成交量（最多+15分）**

```python
量比 > 1.5  → +15分  ⭐⭐⭐
量比 > 1.2  → +10分  ⭐⭐
量比 > 0.8  → +5分   ⭐
量比 ≤ 0.8  → 0分
```

### **维度3：底部位置（最多+10分）**

```python
股价在20日低位30%以内 → +10分  ⭐⭐
股价在20日低位50%以内 → +5分   ⭐
其他位置 → 0分
```

### **维度4：均线（最多+5分）**

```python
MA5 > MA10 且 价格 > MA5 → +5分  ⭐
价格 > MA10 → +3分
其他 → 0分
```

### **1. 简单直接**
- 只有4个维度

- 小涨5分 → 55分 > 40分阈值 ✅
- 中等涨10分+量能5分 → 65分 > 60分阈值 ✅

### **4. 排序有效**
- 涨得多的分数高
- 量能大的分数高
- 底部的分数高
- 排序Top 50能确保质量
