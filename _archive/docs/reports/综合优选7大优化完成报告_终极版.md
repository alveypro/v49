### 【优化3】风险评分维度 ⭐⭐⭐⭐

#### 实现内容

**核心方法**：
`calculate_risk_score()` - 5个风险指标，0-100分风险评分

**5个风险维度**：

1. **波动率风险（30分）**
   - >5%：极高风险（30分）
   - 3-5%：高风险（20分）
   - 2-3%：中等风险（10分）
   - <2%：低风险（0分）

2. **高位风险（25分）**
   - >85%历史高位：极高风险（25分）
   - 70-85%：高位风险（18分）
   - 50-70%：中位风险（10分）
   - <50%：低位安全（0分）

3. **流动性风险（20分）**
   - 成交量<1万手：极差（20分）
   - 成交量<5万手：较差（10分）
   - 成交量>5万手：良好（0分）

4. **历史暴跌风险（15分）**
   - 有跌停历史：高风险（15分）
   - 有大幅下跌（-7%）：中等风险（10分）
   - 历史稳定：低风险（0分）

5. **技术面风险（10分）**
   - 均线空头：高风险（10分）
   - 技术面正常：低风险（0分）

**风险等级**：
- ⚠️ 高风险：≥60分
- 💡 中等风险：30-59分
- ✅ 低风险：<30分

**风险惩罚**：
```python
risk_penalty = risk_score × 0.15  # 风险惩罚系数0.15
final_score = base_score + bonus - risk_penalty
```

**预期效果**：
- 筛除高风险股票（妖股、庄股）
- 夏普比率提升30%
- 最大回撤降低15-20%

## 📈 综合评分公式（最终版）

### 完整评分流程

```python
# 1. 获取市场环境
market_env = get_market_environment()  # 牛市/熊市/震荡市

# 2. 获取动态权重
weights = get_dynamic_weights(market_env)

# 3. 计算6维基础评分
base_scores = {
    'volume_price': 25分,  # 量价配合
    'ma': 20分,            # 均线多头
    'macd': 25分,          # MACD趋势
    'bottom': 15分,        # 底部特征
    'accumulation': 10分,  # 主力吸筹
    'limit': 5分           # 涨停基因
}

# 4. 动态加权
base_score = Σ(score × weight)

# 5. 协同加成（非线性）
synergy_bonus = calculate_synergy_bonus(base_scores)  # 0-20分

# 6. 行业热度加成
industry_bonus = calculate_industry_heat(industry)  # 0-20分

# 7. 风险评分
risk_score = calculate_risk_score(stock_data)  # 0-100分
risk_penalty = risk_score × 0.15

# 8. 最终评分
final_score = base_score + synergy_bonus + industry_bonus - risk_penalty
final_score = max(0, min(100, final_score))
```
