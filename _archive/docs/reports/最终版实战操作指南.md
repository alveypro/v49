## 🎯 止盈止损策略（纪律！）

### 止损规则（必须执行）

```python
【单只股票】
❌ 跌幅达到 -6%：立即止损
❌ 买入后3天未上涨：考虑止损
❌ 出现重大利空：立即止损

【整体仓位】
❌ 整体回撤 > -8%：减仓50%
❌ 整体回撤 > -12%：清仓观望
```

### 止盈规则（保护收益）

```python
【单只股票】
✅ 涨幅达到 +8%：卖出50%
✅ 涨幅达到 +12%：卖出剩余50%
✅ 涨幅达到 +15%：全部卖出

【浮盈回撤】
✅ 从高点回撤 > 5%：减仓50%
✅ 从高点回撤 > 8%：全部卖出

示例：
- 买入价：10元
- 最高涨到：11.5元（+15%）
- 卖出50%后，再涨到12元
- 从12元回撤到11.4元（回撤5%）
  → 立即卖出剩余50%
```

### 持仓时间

```python

✅ 最佳持仓：5-7天
✅ 最长持仓：10天
```

## 🎓 常见错误与避免

### 错误1：忽略板块过滤

```
❌ 错误做法：
- 看到买入价值高就买
- 不管是否萌芽期板块

✅ 正确做法：
1. 先扫描板块，找萌芽期
2. 在AI推荐中筛选这些板块
3. 只买萌芽期板块的股票
```

### 错误2：不执行止损

```
❌ 错误做法：
- 跌了-6%，想等反弹
- 结果越跌越多，-10%甚至-20%

✅ 正确做法：
- -6%立即止损，不犹豫
- 保护本金最重要
- 下次机会还有很多
```

### 错误3：追涨杀跌

```
❌ 错误做法：
- 看到涨了5%，追进去
- 看到跌了2%，割肉离场

✅ 正确做法：
- 只在选股日买入（开盘前决定）
- 按计划执行，不追涨
- 除非止损，不随意卖出
```

### 错误4：仓位过重

```
❌ 错误做法：
- 单只股票10-20%
- 总仓位120%（融资）

✅ 正确做法：
- 单只最多7%
- 总仓位不超过100%
- 留10%备用金
```
