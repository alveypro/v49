## 📊 五大核心系统

### 1️⃣ 市场环境识别系统（Market Regime Analyzer）

#### 功能：智能识别当前市场环境

**五种市场环境**：

```python
1. 稳健牛市 🐮
   - MA5 > MA20 > MA60（多头排列）
   - 20日收益 > 5%
   - 波动率 < 1.5%
   → 策略：追涨有效，趋势为王

2. 波动牛市 🐮⚡
   - MA5 > MA20 > MA60
   - 20日收益 > 3%
   - 波动率较高
   → 策略：波段操作，注意回调

3. 熊市 🐻
   - MA5 < MA20 < MA60（空头排列）
   - 20日收益 < -3%
   → 策略：安全第一，超跌反弹

4. 急跌恐慌 🐻💥
   - 空头排列
   - 20日收益 < -10%
   → 策略：空仓观望，或极度保守

5. 震荡市 📊
   - 均线交织，无明显趋势
   → 策略：潜伏为王，等待突破
```

**实现原理**：
```python
class MarketRegimeAnalyzer:
    def identify_market_regime(self):
        # 获取大盘数据（上证指数/沪深300）
        # 计算MA5、MA20、MA60
        # 计算20日收益率和波动率
        # 根据规则判断环境
        
        if MA5 > MA20 > MA60:
            if return_20d > 5% and volatility < 1.5%:
                return "稳健牛市"
        elif MA5 < MA20 < MA60:
            if return_20d < -10%:
                return "急跌恐慌"
            else:
                return "熊市"
        else:
            return "震荡市"
```

### 2️⃣ 市场情绪计算系统（Sentiment Calculator）

#### 功能：量化市场情绪（-1到+1）

**三个指标**：

```python
1. 涨跌比（权重50%）
   - 涨跌比 > 2: 贪婪
   - 涨跌比 < 0.5: 恐慌
   - 涨跌比 = 1: 中性

2. 涨停/跌停比（权重30%）
   - 涨停 >> 跌停: 极度贪婪
   - 跌停 >> 涨停: 极度恐慌

3. 平均涨跌幅（权重20%）
   - 平均 > 2%: 乐观
   - 平均 < -2%: 悲观
```

**应用**：
```python
# 根据情绪调整选股门槛
if sentiment < -0.7:  # 极度恐慌
    required_score = 80  # 提高门槛，只选最强
elif sentiment > 0.7:  # 极度贪婪
    required_score = 75  # 防止追高
else:
    required_score = 60  # 正常门槛
```

### 3️⃣ 行业轮动分析系统（Industry Rotation Analyzer）

#### 功能：识别热门行业，把握轮动机会

**行业热度计算**：

```python
行业热度 = 
    50% × 5日平均涨幅 / 10 +
    30% × (上涨比例 - 0.5) × 0.6 +
    20% × 涨停股票比例加分

结果：-1（极冷）到 +1（极热）
```

**行业轮动策略**：

```python
热门行业Top3: 加10-7分
热门行业Top5: 加5分
热门行业Top8: 加3分
冷门行业: 提高选股门槛
```

**实际效果**：
- 把握板块轮动机会
- 避开低迷行业
- 预期收益提升：**+5-8%年化**

## 💡 智能风险控制

### 动态止损止盈系统

不再使用固定止损，而是根据多因素智能计算：

```python
止损计算：
基础止损 = 7%

调整因子：
1. 评分高低：
   - 评分 >= 80: 止损放宽到8%
   - 评分 < 60: 止损收紧到5%

2. 市场环境：
   - 熊市：收紧到5%
   - 牛市：放宽到10%

3. 最终止损：
   current_price × (1 - adjusted_stop_loss%)

止盈计算：
基础止盈 = 15%

调整因子：
1. 评分高低：
   - 评分 >= 85: 止盈提高到25%
   - 评分 < 65: 止盈降低到10%

2. 行业热度：
   - 热度 > 0.5: 止盈+5%

3. 最终止盈：
   current_price × (1 + adjusted_take_profit%)
```

```python

evaluator_v7.reset_cache()

        results.append(result)
```
