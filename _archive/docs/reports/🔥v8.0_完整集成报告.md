## 🚀 v8.0 六大革命性功能

### 1. ATR动态风控系统 🛡️
- 根据波动率自动调整止损止盈
- 告别"一刀切"的固定风控

### 2. 三级市场过滤器（择时系统）🌡️
- Level 1: 市场趋势（20日均线）
- Level 2: 市场情绪（涨跌比例）
- Level 3: 市场热度（成交量）
→ 只在好市场中选股！

### 3. 凯利公式仓位管理 💰
- 根据胜率和盈亏比计算最优仓位
- 高胜率+高盈亏比 → 重仓
- 低胜率 → 轻仓

### 4. 18维度高级评分 📊
- v7的8个基础维度
- +10个高级因子（换手率质量、资金集中度等）

### 5. 五星评级系统 ⭐⭐⭐⭐⭐
- 90+ 极力推荐（预期胜率75-80%）
- 80-89 强烈推荐（预期胜率68-75%）
- 70-79 推荐（预期胜率62-68%）

### 6. 动态再平衡系统 🔄
- 自动锁定50%利润
- 持续评分，低于阈值自动换股
