## 🛠️ 修复内容

### 已添加的方法

```python
def _score_safety_margin(self, ind: Dict) -> float:
    """
    【维度7】安全边际（2分）- 🔥极度严格
    
    巅峰版：只给低位股票加分，高位股票扣分
    """
    score = 0
    
    price_pos = ind.get('price_position', 0.5)
    
    # 极度严格：只有真正的低位才加分
    if price_pos < 0.30:  # 极低位（前30%）
        score += 2
    elif price_pos < 0.50:  # 低位（前50%）
        score += 1
    elif price_pos < 0.70:  # 中位（前70%）
        score += 0.5
    # 高位不加分（price_pos >= 0.70）
    
    return min(score, 2)
```

### 评分逻辑

| 价格位置 | 描述 | 得分 | 说明 |
|---------|------|------|------|
| < 30% | 极低位 | 2分 | 最安全的介入位置 |
| 30-50% | 低位 | 1分 | 较安全的介入位置 |
| 50-70% | 中位 | 0.5分 | 中性位置 |
| ≥ 70% | 高位 | 0分 | 追高风险，不加分 |

**设计理念**：巅峰版追求极致安全，只给真正的低位股票加分。
