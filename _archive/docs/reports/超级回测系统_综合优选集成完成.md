```
统计分析：

    - 6维评分平均值
    - AI优化效果
    ↓
UI展示结果
```

## ⚠️ 注意事项

### 1. 计算时间
- 综合优选回测需要计算大量指标
- 预计耗时：30-90秒（取决于样本量）
- 请耐心等待，不要重复点击

### 2. 数据要求
- 需要至少60天的历史数据才能计算指标
- 确保数据库已更新
- 如果数据不足，会跳过该股票

### 3. 市场环境影响
- 回测结果受市场环境影响很大
- 在牛市和熊市中表现差异明显
- 建议查看"市场环境分析"部分

### 4. 过拟合风险
- 6维100分评分是基于技术分析的
- 不是预测未来的魔法
- 实盘使用时需要结合基本面和市场情绪
