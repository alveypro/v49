## 🛠️ 故障排查

### 问题1：市值筛选报错
```
错误: no such column: circ_mv
解决: 数据库缺少市值字段，暂时跳过市值筛选或手动添加
```

### 问题2：筹码数据缺失
```
现象: "筹码分布"维度全部显示10分
影响: 轻微影响，仍可运行
解决: 运行 python3 自动更新筹码分布_混合版.py
```

### 问题3：回测样本过少
```
现象: 只找到<50只股票
原因: 市值范围或评分阈值过严
解决: 扩大市值范围或降低评分阈值
```

### 问题4：回测速度慢
```
现象: 回测超过5分钟
原因: 样本量过大（>1000）
解决: 降低样本数量至500
```
