## 🔥 关键特性

### 1. ATR动态风控
```
买入后自动计算：
- 止损价 = 买入价 - 2.0×ATR
- 止盈价 = 买入价 + 3.0×ATR
- 移动止损 = 最高价 - 1.5×ATR
```

### 2. 三级市场过滤
```
Level 1：趋势（MA20/MA60）
Level 2：情绪（类VIX指数）
Level 3：成交量（资金活跃度）
→ 综合判断是否适合交易
```

### 3. 凯利公式仓位
```
根据历史胜率和盈亏比：
- 自动计算最优仓位
- 5星：25%
- 4星：20%
- 3星：15%
```

### 4. 动态再平衡
```
自动触发：
- 盈利8%：锁定50%利润
- 盈利12%：锁定80%利润
- 评分下降15分：减仓50%
- 评分低于60分：清仓
```
