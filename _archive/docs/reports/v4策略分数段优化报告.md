## 🔍 深度分析

### 1. 为什么60-64分表现最差？

```
问题分析：
- 这个分数段股票质量不够高
- 上涨动力不足
- 下跌风险却不小
- 胜率只有43.4%，期望值为负

结论：必须排除！
```

### 2. 为什么65-69分表现好？

```
关键优势：
- 盈亏比高达1.91（赚得多亏得少）
- 平均盈利5.29%，平均亏损2.78%
- 虽然胜率低(41.4%)，但亏了不多，赚了很多
- 期望值为正0.56%

这是"以小博大"的典型！
```

### 3. 为什么75-79分反而差？

```
可能原因：
- 样本太少(13个)，统计噪音大
- 或者这个分数段恰好遇到不利市场环境
- 需要更多数据验证

建议：谨慎对待
```

### 4. 为什么90+分表现好？

```
合理解释：
- 高分股票确实质量更好
- 胜率50%，平均收益1.35%
- 盈亏比2.03，也很优秀

但问题：信号会很少
```

## 🎯 最终建议

### 推荐使用：**稳健型策略（65-90分）**

**理由**：

1. **数据支持充分**
   - 排除了明确表现差的60-64分
   - 包含了表现优异的65-69分和90+分
   - 总样本数量充足

2. **风险收益平衡**
   - 期望值为正（长期有利）
   - 盈亏比优秀（赚多亏少）
   - 信号数量适中（不会太少）

3. **实战可行性高**
   - 每天能找到足够的推荐
   - 不会因为标准太高而无股可选
   - 适合大部分投资者
