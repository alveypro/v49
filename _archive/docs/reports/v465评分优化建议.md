# v46.5评分系统优化建议

## 🔍 问题诊断

### 当前问题：
1. **评分标准过于严格**：80%的股票得分<40分
2. **平均分只有31分**：远低于预期的50-60分
3. **最高分只有57分**：没有股票能达到60分以上

### 根本原因：
**评分条件太苛刻，大部分维度要求同时满足多个条件**

例如：
- 放量确认（15分）：要求放量1.8倍以上，但大部分股票只有0.6-0.8倍
- 涨幅适中（10分）：要求5日涨幅>3%，但很多股票是下跌的
- 量价配合（10分）：要求放量+上涨，但大部分是缩量或下跌

## 💡 优化方案

### 方案1：降低评分标准（推荐）⭐⭐⭐

**调整各维度的评分条件，让更多股票能得到合理分数**

#### 1. 放量确认（15分）- 降低门槛
```python
# 原来：要求1.8倍才给15分
if 1.8 <= vol_ratio <= 3.0:
    score_volume = 15

# 优化：降低门槛，增加梯度
if 1.5 <= vol_ratio <= 3.0:  # 1.8 → 1.5
    score_volume = 15
elif 1.2 <= vol_ratio <= 3.5:  # 1.5 → 1.2
    score_volume = 12
elif vol_ratio > 1.0:  # 1.3 → 1.0
    score_volume = 8
elif vol_ratio > 0.8:  # 新增：缩量不太严重也给点分
    score_volume = 4
```

#### 2. 涨幅适中（10分）- 允许微涨或横盘
```python
# 原来：要求涨幅>5%才给满分
if 0.05 <= price_chg_5d <= 0.12:
    score_gain = 10

# 优化：降低涨幅要求
if 0.03 <= price_chg_5d <= 0.12:  # 5% → 3%
    score_gain = 10
elif 0.01 <= price_chg_5d <= 0.15:  # 3% → 1%
    score_gain = 8
elif price_chg_5d > -0.01:  # 新增：横盘也给分
    score_gain = 5
elif price_chg_5d > -0.03:  # 新增：微跌也给点分
    score_gain = 2
```

#### 3. 量价配合（10分）- 降低要求
```python
# 原来：要求放量>1.5倍 + 涨幅>2%
if price_chg_5d > 0.02 and vol_ratio > 1.5:
    score_price_vol = 10

# 优化：降低双重要求
if price_chg_5d > 0.01 and vol_ratio > 1.2:  # 2%→1%, 1.5→1.2
    if price_position < 0.3:
        score_price_vol = 10
    elif price_position < 0.5:
        score_price_vol = 7
    else:
        score_price_vol = 4
elif price_chg_5d > 0 and vol_ratio > 1.0:  # 新增：温和放量+微涨
    score_price_vol = 5
elif price_chg_5d > -0.01 and vol_ratio > 0.9:  # 新增：横盘+正常量
    score_price_vol = 3
```

#### 4. 均线多头（8分）- 增加中间档
```python
# 原来：只有完美多头、短期多头、站上20日线三档
if ma5 > ma10 > ma20 and close[-1] > ma5:
    score_ma = 8
elif ma5 > ma10 and close[-1] > ma5:
    score_ma = 6
elif close[-1] > ma20:
    score_ma = 4

# 优化：增加更多档次
if ma5 > ma10 > ma20 and close[-1] > ma5:
    score_ma = 8
elif ma5 > ma10 and close[-1] > ma5:
    score_ma = 6
elif close[-1] > ma20:
    score_ma = 4
elif close[-1] > ma10:  # 新增：站上10日线
    score_ma = 3
elif close[-1] > ma5:
    score_ma = 2
else:  # 新增：即使都不满足也给1分基础分
    score_ma = 1
```

#### 5. 持续放量（7分）- 降低连续天数要求
```python
# 原来：要求连续4天放量才给满分
continuous_days = sum(1 for v in volume[-5:] if v > hist_vol * 1.3)
if continuous_days >= 4:
    score_continuous = 7

# 优化：降低要求
continuous_days = sum(1 for v in volume[-5:] if v > hist_vol * 1.1)  # 1.3→1.1
if continuous_days >= 3:  # 4→3
    score_continuous = 7
elif continuous_days >= 2:
    score_continuous = 5
elif continuous_days >= 1:  # 新增：1天也给分
    score_continuous = 3
```

### 方案2：调整阈值（临时方案）⭐⭐

**不改评分逻辑，只降低筛选阈值**

```python
# 原来的阈值
综合优选：≥60分
底部蓄势：40-59分

# 优化后的阈值
综合优选：≥40分  # 60→40
底部蓄势：30-39分  # 40-59→30-39
```

**优点**：简单快速，不需要改代码
**缺点**：治标不治本，评分仍然不合理

### 方案3：增加基础分（折中方案）⭐⭐⭐

**给所有股票一个基础分，避免大量0分维度**

```python
# 在每个维度增加最低分
# 例如：放量确认
score_volume = 3  # 基础分3分
if 1.5 <= vol_ratio <= 3.0:
    score_volume = 15
elif 1.2 <= vol_ratio <= 3.5:
    score_volume = 12
elif vol_ratio > 1.0:
    score_volume = 8
elif vol_ratio > 0.8:
    score_volume = 5
# 即使缩量也有3分基础分
```

## 🎯 推荐实施方案

**采用方案1 + 方案3的组合**：

1. **降低各维度的评分门槛**（方案1）
2. **给每个维度设置最低基础分**（方案3）
3. **调整阈值**：
   - 综合优选：≥50分（原60分）
   - 底部蓄势：35-49分（原40-59分）

### 预期效果：
- 平均分提升到：**45-55分**（原31分）
- 40分以上股票：**50-60%**（原20%）
- 60分以上股票：**10-20%**（原0%）

## ⚠️ 注意事项

1. **不要过度降低标准**：
   - 评分太容易会失去筛选意义
   - 建议保持一定的区分度

2. **保留核心逻辑**：
   - 底部位置、放量确认等核心维度仍然重要
   - 只是降低门槛，不是取消条件

3. **测试验证**：
   - 修改后重新运行诊断脚本
   - 确保评分分布合理

4. **用户体验**：
   - 让用户能找到40-59分的"蓄势股"
   - 让用户能找到60分以上的"优质股"
   - 避免"没有符合条件的股票"的尴尬
