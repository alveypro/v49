# Tushare 基础接口速查

以下内容来自“基础信息.docx”的要点整理，便于系统配置与排障。

## 1) 股票基础信息
- 接口：`stock_basic`
- 用途：股票代码、名称、行业、上市日期等基础信息
- 建议：全量拉取一次，落库后复用

示例（筛选正常上市）：
- `pro.stock_basic(exchange='', list_status='L', fields='ts_code,symbol,name,area,industry,list_date')`

## 2) 交易日历
- 接口：`trade_cal`
- 用途：判断交易日/非交易日（交易日历权威来源）
- 建议：用于数据中心“最新交易日”校验

示例：
- `pro.trade_cal(exchange='SSE', start_date='20240101', end_date='20241231', is_open='1')`

## 3) 沪深港通成分
- 接口：`hs_const`
- 用途：识别沪股通/深股通标的
- 示例：`pro.hs_const(hs_type='SH')` / `pro.hs_const(hs_type='SZ')`

## 4) 上市公司基础信息
- 接口：`stock_company`
- 用途：公司信息、经营范围、注册资本等

## 5) 管理层信息
- 接口：`stk_managers`
- 用途：高管任职与履历信息

## 6) 管理层薪酬与持股
- 接口：`stk_rewards`

## 7) IPO 新股列表
- 接口：`new_share`

## 8) 指数数据（v8 需要）
- 接口：`index_daily`
- 用途：指数行情（例如 `000001.SH`）
- 说明：需写入 `daily_trading_data` 供 v8 使用

## 常见问题
- 若出现 `api.waditu.com` 连接失败，请先解决外网/代理问题再拉取数据。
